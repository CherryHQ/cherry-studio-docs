# 反馈 & 建议

## GitHub Issues

{% hint style="info" %}
**推荐 Bug 提交与功能建议渠道**。在这里提交能防止被开发者遗忘，还可以参与社区深度讨论。

* **如果你发现了任何的 Bug**，或者对产品有新的构想与建议，可以直接在这里提交反馈：
