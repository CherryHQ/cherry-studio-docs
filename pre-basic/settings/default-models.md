# 默认模型设置

Cherry Studio 在很多场景下都需要"挑一个模型用一下"（比如帮你给对话起名字、优化提示词、做翻译），不可能每次都问你"用哪个模型"。**默认模型设置**就是告诉 Cherry Studio：**当我没明说时，用哪个模型**。

> 注意：这些是"幕后小工"用的模型，**和你聊天用的模型可以不一样**。聊天主模型是在每个助手里单独设的。

## 4 个默认模型分别管什么？

### 默认助手模型

* **谁用**：你新建一个助手但没指定模型时，自动用这个
* **谁还用**：对话框右上角的"优化提示词"、[划词助手](../../cherrystudio/preview/selection-assistant.md)
* **怎么选**：选一个你常用、稳定、价格合理的模型（比如 GPT-4o-mini、DeepSeek-V3、Claude Haiku）

### 话题命名模型

* **谁用**：每开一段新对话，Cherry Studio 会自动给它起个简短标题（比如"重构 React 组件"），就是用这个模型完成的
* **怎么选**：用**便宜快速**的模型就行（GPT-4o-mini、DeepSeek-V3、bge 等），起名字不需要顶级智能

### 翻译模型

* **谁用**：对话框/绘画/翻译页面里"一键翻译"功能
* **怎么选**：翻译任务对模型要求中等，普通对话模型都行；如果中英互译多，DeepSeek 或 Claude 系列效果好

### 快捷助手模型

* **谁用**：[快捷助手](../../cherrystudio/preview/kuai-jie-zhu-shou.md)（全局快捷键唤起的迷你窗口）
* **怎么选**：**响应快**的模型优先，因为你是急用：GPT-4o-mini / DeepSeek-V3 都不错

## 一句话推荐

如果你不想细抠，照下面填就行：

| 字段 | 推荐 |
|---|---|
| 默认助手模型 | 你最常用的对话模型 |
| 话题命名模型 | 便宜快速的（如 `gpt-4o-mini`、`deepseek-chat`）|
| 翻译模型 | 任何对话模型都行 |
| 快捷助手模型 | 响应快的（如 `gpt-4o-mini`、`claude-haiku-4`）|

不知道选什么的话，全部默认就好，用一段时间发现哪个不够用再回来调整。
