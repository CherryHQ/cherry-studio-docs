# Структура кода

```yaml
...
├─ docs/ # документация
├─ resources/ # ресурсы
├─ scripts/ # скрипты
└─ src/ # основной исходный код
    ├─main/ # главный процесс
    ├─preload/ # презагрузка
    └─renderer/ # рендер процесс
        ├─src/ # исходный код рендер процесса
            ├─assets/ # ресурсы
                ├─fonts/ # шрифты
                ├─images/ # изображения (аватарки и др.)
                └─styles/ # стили
            ├─components/ # компоненты
            ├─config/ # конфигурация
            ├─context/ # контекст
            ├─databases/ # базы данных
            ├─hooks/ # кастомные хуки
            ├─i18n/ # интернационализация
            ├─pages/ # страницы
            ├─providers/ # провайдеры
            ├─services/ # сервисы
            ├─store/ # хранилище состояний
            ├─types/ # типы TypeScript
            ├─utils/ # утилиты
            ...
        ...
    ...
...
```
