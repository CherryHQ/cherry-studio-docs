# Эффективные методы задавания вопросов

Cherry Studio — это бесплатный проект с открытым исходным кодом. По мере роста проекта нагрузка команды разработчиков увеличивается. Чтобы снизить коммуникационные издержки и максимально эффективно решать ваши вопросы, мы просим перед обращением следовать этим шагам — это позволит команде сосредоточиться на разработке и поддержке проекта. Благодарим за понимание!

## 1. Поиск в документации

Большинство базовых вопросов решаются внимательным изучением документации:

* Функциональные вопросы и инструкции по использованию ПО можно найти в разделе [功能介绍](../cherrystudio/preview/);
* Часто задаваемые вопросы собраны на странице [常见问题](questions.md) — сначала проверьте наличие решения там;
* Для сложных вопросов воспользуйтесь поиском или задайте вопрос в поисковой строке;
* Внимательно читайте подсказки в документации — они помогут избежать многих проблем;
* Изучите Issues на GitHub ([ссылка](https://github.com/CherryHQ/cherry-studio/issues)) на наличие схожих вопросов и решений.

## 2. Поиск в сети и консультация с ИИ

Вопросы, не связанные с функционалом клиента (например, ошибки моделей, некорректные ответы, настройка параметров), рекомендуется сначала искать в сети или описывать проблему ИИ для поиска решений.
