Руководство по использованию голосовых функций Cherry Studio

## I. Обзор голосовых функций

Cherry Studio предоставляет три основные голосовые функции: TTS (Text-to-Speech, синтез речи), ASR (Automatic Speech Recognition, распознавание речи) и голосовые вызовы. Эти функции позволяют вам естественно общаться с ИИ с помощью голоса, улучшая пользовательский опыт.

- **TTS (Синтез речи)**: преобразует текстовые ответы ИИ в голосовое воспроизведение
- **ASR (Распознавание речи)**: преобразует ваш голос в текстовый ввод
- **Голосовые вызовы**: сочетание TTS и ASR для голосового диалога по аналогии с ChatGPT

## II. Функция TTS (Text-to-Speech)

### 1. Поддерживаемые сервисы

Cherry Studio поддерживает четыре типа TTS-сервисов:

- **OpenAI**: использует TTS API от OpenAI, требуется API-ключ
- **Браузерный TTS**: использует встроенные функции синтеза речи браузера, бесплатно без настройки
- **Siliconflow**: использует TTS-сервис Siliconflow, требуется API-ключ
- **Бесплатный онлайн TTS**: использует бесплатные онлайн-TTS сервисы, не требует API-ключа

### 2. Настройка

1. Перейдите в раздел настроек, выберите вкладку "Голосовые функции"
2. Во вкладке "TTS":
   - Активируйте функцию TTS (переключите тумблер)
   - Выберите тип TTS-сервиса
   - В зависимости от выбранного сервиса настройте параметры:
     - OpenAI: укажите API-ключ, адрес API, выберите голос и модель
     - Браузерный TTS: выберите голос
     - Siliconflow: укажите API-ключ, адрес API, выберите голос, модель, формат ответа и скорость
     - Бесплатный онлайн TTS: выберите голос и формат вывода
3. Настройте фильтры TTS (опционально):
   - Фильтрация процесса мышления
   - Фильтрация Markdown-разметки
   - Фильтрация блоков кода
4. Установите отображение индикатора прогресса TTS
5. Нажмите кнопку "Тест TTS" для проверки конфигурации

### 3. Использование

- После активации TTS ответы ИИ автоматически преобразуются в речь
- В интерфейсе чата под каждым ответом ИИ отображается кнопка воспроизведения TTS
- Нажмите кнопку для воспроизведения/паузы
- Если включен индикатор прогресса, под текстом отображается прогресс воспроизведения
- Длинные тексты автоматически сегментируются и воспроизводятся последовательно

## III. Функция ASR (Распознавание речи)

### 1. Поддерживаемые сервисы

Cherry Studio поддерживает три типа ASR-сервисов:

- **OpenAI**: использует модель Whisper от OpenAI, требуется API-ключ
- **Браузер**: использует встроенное распознавание речи браузера, бесплатно без настройки
- **Локальный сервер**: подключение к локальному WebSocket-серверу для распознавания

### 2. Настройка

1. Перейдите в раздел настроек, выберите вкладку "Голосовые функции"
2. Во вкладке "ASR":
   - Активируйте функцию ASR (переключите тумблер)
   - Выберите тип ASR-сервиса
   - В зависимости от выбранного сервиса настройте параметры:
     - OpenAI: укажите API-ключ, адрес API, выберите модель
     - Браузер: дополнительная настройка не требуется
     - Локальный сервер: установите автоматический запуск ASR-сервера при старте приложения
   - Выберите язык распознавания (по умолчанию китайский)
3. Нажмите кнопку "Тест ASR" для проверки конфигурации

### 3. Использование

- После активации ASR рядом с полем ввода появляется кнопка распознавания речи
- Нажмите кнопку для начала записи
- После речи она преобразуется в текст и вставляется в поле ввода
- Повторное нажатие останавливает запись
- Поддерживается непрерывное распознавание нескольких фраз в режиме накопления

## IV. Функция голосовых вызовов

### 1. Особенности

- Сочетание TTS и ASR для голосового диалога
- Плавающее перетаскиваемое окно
- Поддержка режима "удерживай для разговора"
- Настройка горячих клавиш
- Сворачивание окна
- Выбор специализированной модели для вызовов
- Кастомизация промптов

### 2. Настройка

1. Перейдите в раздел настроек, выберите вкладку "Голосовые функции"
2. Во вкладке "Функция вызовов":
   - Активируйте голосовые вызовы (переключите тумблер)
   - Нажмите "Выбрать модель" для выбора ИИ-модели
   - Настройте промпты для вызовов (опционально)
   - Нажмите "Сохранить" или "Сброс" для применения/восстановления промптов

### 3. Использование

1. В интерфейсе чата нажмите иконку телефона справа от поля ввода
2. Откроется окно вызова с приветственным сообщением
3. Удерживайте кнопку "Удерживай для разговора" (или используйте горячую клавишу)
4. Отпустите кнопку для отправки записи ИИ
5. Ответ ИИ воспроизводится через TTS
6. Управляйте через кнопки:
   - Иконка микрофона: отключение/включение TTS
   - Иконка паузы: приостановка/возобновление
   - Иконка настроек: конфигурация горячих клавиш
   - Иконка сворачивания: минимизация окна
7. Нажмите иконку закрытия для завершения вызова

### 4. Настройка горячих клавиш

1. В окне вызова нажмите иконку настроек
2. В панели настроек нажмите кнопку горячих клавиш
3. Нажмите нужную клавишу (например, пробел, Shift)
4. Нажмите "Сохранить"
5. Используйте: удерживайте клавишу для записи, отпустите для отправки

## V. Частые проблемы и решения

### 1. Проблемы с TTS

- **Нет звука**: проверьте активацию TTS, правильность типа сервиса и параметров
- **Плохое качество**: попробуйте другой тип сервиса или голос
- **Ошибки воспроизведения**: проверьте API-ключ и интернет-соединение

### 2. Проблемы с ASR

- **Речь не распознаётся**: проверьте активацию ASR и параметры
- **Низкая точность**: смените сервис или настройте микрофон
- **Ошибка подключения**: проверьте локальный сервер или перезапустите приложение

### 3. Проблемы с вызовами

- **Окно не открывается**: проверьте активацию и настройки TTS/ASR
- **Не реагирует на удержание**: проверьте разрешения микрофона
- **Нет голосового ответа**: проверьте активацию TTS и режим "без звука"

## VI. Расширенные настройки

### 1. Расширенные настройки TTS
- Фильтры: оптимизация воспроизведения
- Индикатор прогресса: отображение прогресса
- Пользовательские голоса: добавление кастомных вариантов

### 2. Расширенные настройки ASR
- Автозапуск сервера: опция при старте приложения
- Выбор языка: смена языка распознавания

### 3. Расширенные настройки вызовов
- Кастомные промпты: адаптация ответов под голосовое общение
- Выделенная модель: отдельная модель для вызовов
- Горячие клавиши: кастомизация управления записью

## VII. Рекомендации по использованию

### 1. Выбор TTS-сервиса
- Для высокого качества: OpenAI или Siliconflow
- Для простоты: браузерный или бесплатный онлайн TTS

### 2. Выбор ASR-сервиса
- Для высокой точности: OpenAI
- Для простоты: браузерный вариант

### 3. Оптимизация вызовов
- Используйте наушники для предотвращения эха
- Обеспечьте тихую обстановку для повышения точности
- Адаптируйте промпты под голосовое общение

### 4. Адаптация настроек
- Для текстового общения: только TTS
- Для голосового ввода: только ASR
- Для полного голосового общения: активируйте вызовы

Надеемся, это руководство поможет вам полностью раскрыть потенциал голосовых функций Cherry Studio для более естественного и удобного взаимодействия с ИИ!
