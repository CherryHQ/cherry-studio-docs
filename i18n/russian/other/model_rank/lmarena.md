## Описание

- **Ранг (UB)**: Рейтинг, рассчитанный на основе модели Брэдли-Терри. Этот рейтинг отражает общую производительность модели на арене и оценивает **верхнюю границу** её рейтинга Elo для понимания потенциал极а модели.
- **Ранг (StyleCtrl)**: Рейтинг с применением контроля стиля диалога. Устраняет предвзятость, вызванную стилем ответов моделей (например, подробность/краткость), для более чистой оценки базовых способностей моделей.
- **Модель**: Название большой языковой модели (LLM). Каждая модель содержит ссылку для перехода.
- **Оценка**: Рейтинг Elo, полученный моделью через пользовательские голоса на арене. Это относительная система оценки - чем выше значение, тем лучше производительность. Значение динамически изменяется, отражая текущую конкуренцию.
- **Диап. точн.**: 95% доверительный интервал рейтинга Elo (например: `+6/-6`). Чем меньше диапазон, тем стабильнее и надежнее рейтинг; более широкий интервал указывает на недостаток данных или вариативность производительности. Количественная оценка точности рейтинга.
- **Голоса**: Общее ко极личество голосов, полученных моделью на арене. Большее количество голосов обычно означает более высокую статистическую надежность рейтинга.
- **Провайдер**: Организация или компания, предоставляющая модель.
- **Лицензия**: Тип лицензии модели, например: проприетарная (Proprietary), Apache 2.0, MIT и др.
- **Актуальность**: Дата последнего обновления знаний, использованных при обучении модели. **Нет данных** означает, что информация отсутствует или неизвестна.

## Источник данных и частота обновлений

Данные для этого лидерборда автоматически генерируются пр极оектом [fboulnois/llm-leaderboard-csv](https://github.com/fboulnois/llm-leaderboard-csv), который собирает и обрабатывает данные с [lmarena.ai](https://lmarena.ai/). Лидерборд ежедневно обновляется с помощью GitHub Actions.

## Отказ от ответственности

Этот отчет предназначен исключительно для информационных целей. Данные в лидерборде динамически изменяются и основаны на пользовательских предпочтениях, выраженных через голосование в Chatbot Arena за определенный период. Полнота и точность данных зависят от исходного источника  `fboulnois/llm-leaderboard-csv`. Модели могут распространяться 极под разными лицензиями - при использовании обязательно обращайтесь к официальной документации поставщиков моделей.
