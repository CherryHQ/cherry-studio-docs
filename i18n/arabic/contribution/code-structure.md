# هيكل الكود

```yaml
...
├─ docs/ # دليل المستندات
├─ resources/ # دليل ملفات الموارد
├─ scripts/ # دليل ملفات السكريبتات
└─ src/ # دليل شفرة المصدر الرئيسية
    ├─main/ # كود العملية الرئيسية
    ├─preload/ # دليل سكريبتات التحميل المسبق
    └─renderer/ # كود عملية التصيير
        ├─src/ # شفرة المصدر لعملية التصيير
            ├─assets/ # ملفات الموارد
                ├─fonts/ # ملفات الخطوط
                ├─images/ # ملفات الصور (مثل الصورة الرمزية)
                └─styles/ # ملفات الأنماط
            ├─components/ # المكونات
            ├─config/ # ملفات التكوين
            ├─context/ # السياق
            ├─databases/ # ملفات متعلقة بقواعد البيانات
            ├─hooks/ # خطافات مخصصة (Hooks)
            ├─i18n/ # ملفات التدويل
            ├─pages/ # ملفات الصفحات
            ├─providers/ # تكوينات مزودي الخدمة
            ├─services/ # ملفات الخدمة
            ├─store/ # ملفات إدارة الحالة
            ├─types/ # ملفات تعريف أنواع TypeScript
            ├─utils/ # دوال الأدوات
            ...
        ...
    ...
...
```
