# アシスタントサブスクリプション設定


{% hint style="warning" %}
このドキュメントはAIによって中国語から翻訳されており、まだレビューされていません。
{% endhint %}




アシスタントサブスクリプションのリンクを変更することで、アシスタントライブラリ内のアシスタントテンプレートをすばやく切り替えることができます。
