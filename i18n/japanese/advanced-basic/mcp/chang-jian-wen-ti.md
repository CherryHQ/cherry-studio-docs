# よくある質問

### 1. mcp-server-time

<figure><img src="../../.gitbook/assets/telegram-cloud-photo-size-5-6068931438453048569-y.jpg" alt=""><figcaption><p>エラースクリーンショット</p></figcaption></figure>

**解決方法**  

「パラメータ」欄に以下のように入力してください：

```
mcp-server-time
--local-timezone
<あなたの標準タイムゾーン、例: Asia/Shanghai>
```
