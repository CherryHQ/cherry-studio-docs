# 음성 기능

```
Cherry Studio 음성 기능 사용 설명서

## 1. 음성 기능 개요

Cherry Studio는 TTS(텍스트 음성 변환), ASR(음성 인식), 음성 통화 등 세 가지 주요 음성 기능 모듈을 제공합니다. 이를 통해 사용자는 음성으로 AI와 자연스러운 대화를 할 수 있어 사용 경험이 향상됩니다.

- TTS(텍스트 음성 변환): AI 응답 텍스트를 음성으로 출력
- ASR(음성 인식): 사용자의 음성을 텍스트 입력으로 변환
- 음성 통화: TTS와 ASR을 결합하여 ChatGPT와 유사한 음성 대화 경험 제공

## 2. TTS(텍스트 음성 변환) 기능

### 2.1 지원 서비스 유형

Cherry Studio는 네 가지 TTS 서비스 유형을 지원합니다:

- OpenAI: OpenAI TTS API 사용, API 키 필요
- 브라우저 TTS: 브라우저 내장 음성 합성 기능 사용, 무료 설정 불필요
- Siliconflow(실리콘플로우): Siliconflow TTS 서비스 사용, API 키 필요
- 무료 온라인 TTS: 무료 온라인 TTS 서비스 사용, API 키 불필요

### 2.2 설정 방법

1) 설정 페이지에서 "음성 기능" 탭 선택
2) "TTS" 서브 탭에서:
   - TTS 기능 활성화(토글 스위치 켬)
   - TTS 서비스 유형 선택
   - 선택한 서비스 유형에 따라 매개변수 구성:
     - OpenAI: API 키, API 주소 입력, 음성 및 모델 선택
     - 브라우저 TTS: 음성 선택
     - Siliconflow: API 키, API 주소 입력, 음성/모델/응답 형식/말속도 선택
     - 무료 온라인 TTS: 음성 및 출력 형식 선택
3) TTS 필터 옵션 구성(선택적):
   - 사고 과정 필터링
   - Markdown 마크업 필터링
   - 코드 블록 필터링
4) TTS 진행 표시줄 표시 여부 설정
5) "테스트 TTS" 버튼 클릭하여 구성 확인

### 2.3 사용 방법

- TTS 기능 활성화 시 AI 응답이 자동으로 음성으로 출력됨
- 채팅 인터페이스에서 각 AI 응답 아래 TTS 재생 버튼 표시
- 재생 버튼 클릭으로 음성 재생/일시정지
- TTS 진행 표시줄 활성화 시 텍스트 아래 재생 진행률 표시
- 장문 텍스트는 자동 분할 합성 후 연속 재생

## 3. ASR(음성 인식) 기능

### 3.1 지원 서비스 유형

Cherry Studio는 세 가지 ASR 서비스 유형 지원:

- OpenAI: OpenAI의 Whisper 모델 사용, API 키 필요
- 브라우저: 브라우저 내장 음성 인식 기능 사용, 무료 설정 불필요
- 로컬 서버: 로컬 WebSocket 서버 연결 사용

### 3.2 설정 방법

1) 설정 페이지에서 "음성 기능" 탭 선택
2) "ASR" 서브 탭에서:
   - ASR 기능 활성화(토글 스위치 켬)
   - ASR 서비스 유형 선택
   - 선택한 서비스 유형에 따라 매개변수 구성:
     - OpenAI: API 키, API 주소 입력, 모델 선택
     - 브라우저: 추가 구성 불필요
     - 로컬 서버: 앱 시작 시 ASR 서버 자동 시작 여부 설정 가능
   - 음성 인식 언어 선택(기본값: 중국어)
3) "테스트 ASR" 버튼 클릭하여 구성 확인

### 3.3 사용 방법

- ASR 기능 활성화 시 입력창 옆에 음성 인식 버튼 표시
- 음성 인식 버튼 클릭으로 녹음 시작
- 발언 후 음성이 텍스트로 변환되어 입력창에 표시
- 버튼 재클릭으로 녹음 종료
- 음성 인식은 다중 문장 연속 인식 지원(누적 모드)

## 4. 음성 통화 기능

### 4.1 기능 특징

- TTS와 ASR 결합으로 ChatGPT 유사 음성 대화 경험 제공
- 드래그 가능한 플로팅 창 인터페이스
- 길게 눌러 말하기 모드 지원
- 사용자 정의 단축키 지원
- 창 접기 지원
- 전용 음성 통화 모델 선택 가능
- 사용자 정의 프롬프트 지원

### 4.2 설정 방법

1) 설정 페이지에서 "음성 기능" 탭 선택
2) "통화 기능" 서브 탭에서:
   - 음성 통화 기능 활성화(토글 스위치 켬)
   - "모델 선택" 버튼 클릭하여 음성 통화용 AI 모델 선택
   - 프롬프트 입력창에 사용자 정의 음성 통화 프롬프트 작성(선택적)
   - "저장" 버튼으로 프롬프트 저장 또는 "재설정" 버튼으로 기본값 복원

### 4.3 사용 방법

1) 채팅 인터페이스에서 입력창 오른쪽 음성 통화 버튼(전화 아이콘) 클릭
2) 음성 통화 창 열림, 환영 음성 재생
3) "길게 눌러 말하기" 버튼 길게 눌러 녹음 시작(또는 설정한 단축키 사용)
4) 버튼 놓아 녹음 종료 및 AI 처리
5) AI가 응답 생성 후 TTS로 재생
6) 창 내 컨트롤 버튼 사용:
   - 음소거/해제 버튼: TTS 출력 제어
   - 일시정지/계속 버튼: 대화 일시중지/재개
   - 설정 버튼: 단축키 구성
   - 접기 버튼: 창 축소(길게 눌러 말하기 행만 남김)
7) 닫기 버튼 클릭으로 통화 종료

### 4.4 단축키 설정

1) 음성 통화 창에서 설정 버튼 클릭
2) 설정 패널에서 단축키 버튼 클릭
3) 원하는 키 설정(스페이스바, Shift 키 등)
4) "저장" 버튼으로 설정 저장
5) 사용 시 설정한 단축키 길게 눌러 녹음 시작, 놓으면 종료 및 전송

## 5. 자주 묻는 질문 및 해결책

### 5.1 TTS 관련 문제

- 문제: TTS 소리 출력 불가
  해결책: TTS 기능 활성화 확인, 올바른 서비스 유형 선택 및 필수 매개변수 구성 확인

- 문제: TTS 재생 품질 불량
  해결책: 다른 TTS 서비스 유형 또는 음성 변경 시도

- 문제: TTS 재생 시 오류 메시지 표시
  해결책: API 키 정확성 확인, 네트워크 연결 상태 점검

### 5.2 ASR 관련 문제

- 문제: ASR 음성 인식 불가
  해결책: ASR 기능 활성화 확인, 올바른 서비스 유형 선택 및 필수 매개변수 구성 확인

- 문제: ASR 인식 정확도 낮음
  해결책: 다른 ASR 서비스 유형 시도 또는 마이크 위치/볼륨 조정

- 문제: ASR 서버 연결 실패
  해결책: 로컬 서버 실행 상태 확인 또는 앱 재시작 시도

### 5.3 음성 통화 관련 문제

- 문제: 음성 통화 창 열기 불가
  해결책: 음성 통화 기능 활성화 확인, TTS/ASR 기능 구성 정확성 점검

- 문제: 길게 눌러 말하기 반응 없음
  해결책: 마이크 권한 부여 확인 또는 음성 통화 재시작 시도

- 문제: AI 응답 음성 출력 없음
  해결책: TTS 기능 활성화 확인, 음소거 상태 확인

## 6. 고급 설정 및 사용자 정의 옵션

### 6.1 TTS 고급 설정

- 필터 옵션: 사고 과정/Markdown 마크업/코드 블록 필터링 선택 가능(TTS 재생 원활화)
- 진행 표시줄: TTS 진행률 표시 여부 선택
- 사용자 정의 음성/모델: 사용자 정의 음성 및 모델 옵션 추가 가능

### 6.2 ASR 고급 설정

- 서버 자동 시작: 앱 시작 시 ASR 서버 자동 시작 여부 설정
- 언어 선택: 다양한 음성 인식 언어 선택

### 6.3 음성 통화 고급 설정

- 사용자 정의 프롬프트: 음성 통화 모드용 AI 응답 방식 지시 가능
- 전용 모델 선택: 현재 대화 모델과 별개로 음성 통화 전용 AI 모델 선택
- 단축키 사용자 정의: 녹음 제어용 사용자 정의 단축키 설정

## 7. 사용 제안

1. 적합한 TTS 서비스 선택:
   - 고품질 음성: OpenAI 또는 Siliconflow 추천
   - API 구성 없음: 브라우저 TTS 또는 무료 온라인 TTS 사용

2. 적합한 ASR 서비스 선택:
   - 고정확도: OpenAI 추천
   - API 구성 없음: 브라우저 내장 음성 인식 사용

3. 음성 통화 경험 최적화:
   - 헤드폰 사용 시 TTS 출력 재수신 방지
   - 조용한 환경에서 사용 시 인식 정확도 향상
   - 사용자 정의 프롬프트로 음성 재생에 적합한 AI 응답 유도

4. 요구사항에 맞춰 설정 조정:
   - 주로 텍스트 교환: TTS 기능만 활성화
   - 주로 음성 입력: ASR 기능만 활성화
   - 완전한 음성 대화 경험: 음성 통화 기능 활성화

본 사용 설명서가 Cherry Studio 음성 기능을 최대한 활용하고, 더 자연스럽고 편리한 AI 상호작용 경험을 즐기시는 데 도움이 되길 바랍니다!
```
