# Häufig gestellte Fragen

### 1. mcp-server-time

**Lösung**

Tragen Sie im Abschnitt "Parameter" Folgendes ein:

```
mcp-server-time
--local-timezone
<Ihre Standardzeitzone, z. B.: Asia/Shanghai>
```
