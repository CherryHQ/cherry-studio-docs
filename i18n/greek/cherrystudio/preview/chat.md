## Βοηθοί και Θέματα

### Βοηθοί

Ο `Βοηθός` είναι μια προσαρμοσμένη διαμόρφωση του επιλεγμένου μοντέλου με ρυθμίσεις όπως προκαθορισμένα prompt και παραμέτρους, ώστε το μοντέλο να ανταποκρίνεται καλύτερα στις αναμενόμενες εργασίες σας.

Ο `Προεπιλεγμένος Βοηθός Συστήματος` περιλαμβάνει γενικές παραμέτρους (χωρίς prompt) που μπορείτε να χρησιμοποιήσετε απευθείας ή να αναζητήσετε προκαθορισμένες ρυθμίσεις στη [σελίδα Πράκτορων](agents.md).

### Θέματα

Ο `Βοηθός` είναι το γονικό σύνολο των `Θεμάτων`. Κάθε βοηθός μπορεί να δημιουργήσει πολλαπλά θέματα (διαλόγους), όπου όλα τα `Θέματα` μοιράζονται τις ρυθμίσεις του βοηθού (παραμέτρους και προκαθορισμένα prompts).

## Ρυθμίσεις Βοηθού

Επιλογή βοηθού → <mark style="background-color:yellow;">Μενού δεξιού κλικ</mark> → Ρυθμίσεις

### Επεξεργασία Βοηθού

{% hint style="info" %}
Οι ρυθμίσεις ισχύουν για όλα τα θέματα του βοηθού.
{% endhint %}

<figure><img src="../../.gitbook/assets/image (6) (1) (1).png" alt=""><figcaption></figcaption></figure>

#### Ρυθμίσεις Prompt

#### <mark style="color:blue;">**`Όνομα`**</mark>：

Προσαρμοσμένο αναγνωρίσιμο όνομα βοηθού.

#### <mark style="color:blue;">**`Prompt`**</mark>：

Παραπομπή προς τη σελίδα Πράκτορων για σύνταξη.

#### Ρυθμίσεις Μοντέλου

#### <mark style="color:blue;">**`Προεπιλεγμένο Μοντέλο`**</mark>：

Καθορισμός προεπιλεγμένου μοντέλου για τον βοηθό. Χωρίς ρύθμιση, χρησιμοποιείται το καθολικό προεπιλεγμένο.

{% hint style="info" %}
Προτεραιότητα: Προεπιλογή βοηθού > Καθολική προεπιλογή ([Προεπιλεγμένο Μοντέλο Βοηθού](settings/default-models.md#mo-ren-zhu-shou-mo-xing)).
{% endhint %}

#### <mark style="color:blue;">**`Αυτόματη Επαναφορά Μοντέλου`**</mark>：

Ενεργό: Το νέο θέμα χρησιμοποιεί το προεπιλεγμένο μοντέλο του βοηθού, ακόμα κι αν έχει αλλάξει σε προηγούμενο θέμα.<br>
Ανενεργό: Το νέο θέμα διατηρεί το τελευταίο χρησιμοποιημένο μοντέλο.

#### <mark style="color:blue;">**`Θερμοκρασία (Temperature)`**</mark> ：

Έλεγχος δημιουργικότητας (Προεπιλογή: 0.7):
* **Χαμηλό (0-0.3)**: Ακριβές, για κώδικα/δεδομένα
* **Μέτριο (0.4-0.7)**: Ισορροπημένο, για συζητήσεις
* **Υψηλό (0.8-1.0)**: Δημιουργικό, για συγγραφή

#### <mark style="color:blue;">**`Top P (Δειγματοληψία Πυρήνα)`**</mark>：

Προεπιλογή: 1. Όσο μικρότερη, τόσο πιο προβλέψιμο:
* **Χαμηλό (0.1-0.3)**: Προσεγμένη επιλογή λέξεων
* **Μέτριο (0.4-0.6)**: Ισορροπημένη ποικιλία
* **Υψηλό (0.7-1.0)**: Πλούσια γλωσσική ποικιλία

{% hint style="info" %}
- Χρησιμοποιείτε ανεξάρτητα ή συνδυαστικά
- Πειραματιστείτε με βάση τις ανάγκες σας
- Οι προτεινόμενες τιμές μπορεί να διαφέρουν ανά μοντέλο
{% endhint %}

#### <mark style="color:blue;">**`Περίγραμμα (Context Window)`**</mark>

Αριθμός μηνυμάτων για διατήρηση πλαισίου. Περισσότερα μηνύματα = περισσότερα Token:
* **5-10**: Ιδανικό για συζητήσεις
* **\>10**: Σύνθετες εργασίες (π.χ. συγγραφή)

#### <mark style="color:blue;">**`Περιορισμός Μήκους (MaxToken)`**</mark>

Μέγιστο [Token](https://docs.cherry-ai.com/question-contact/knowledge#shen-me-shi-tokens) ανά απάντηση. Τυπικά όρια: 32k (συχνά), 64k ή περισσότερα.

> Παράδειγμα: Για έλεγχο σύνδεσης μοντέλου, ορίστε MaxToken=1.

{% hint style="success" %}
Προτάσεις:
* Σύνολα: 500-800
* Σύντομα κείμενα: 800-2000
* Κώδικας: 2000-3600
* Μεγάλα κείμενα: 4000+ (απαιτεί υποστήριξη μοντέλου)
{% endhint %}

{% hint style="warning" %}
Υψηλές τιμές μπορεί να προκαλέσουν περικοπές ή ημιτελείς απαντήσεις.
{% endhint %}

#### <mark style="color:blue;">**`Ροή (Stream)`**</mark>

Ενεργό: Απάντηση "γράφει" αυτόματα (αποτέλεσμα γραφομηχανής).<br>
Ανενεργό: Επιστροφή πλήρους απάντησης ταυτόχρονα.

{% hint style="info" %}
Απενεργοποιήστε για μοντέλα χωρίς υποστήριξη ροής (π.χ. o1-mini).
{% endhint %}

#### <mark style="color:blue;">**`Προσαρμοσμένες Παράμετροι`**</mark>

Προσθήκη επιπλέον παραμέτρων στο αίτημα (π.χ. `presence_penalty`). Προχωρημένη χρήση.

> Παραδείγματα: top-p, max_tokens, stream.

Σύνταξη: Όνομα-Τύπος-Τιμή. Ανατρέξτε σε: [Τεκμηρίωση](https://openai.apifox.cn/doc-3222739)

{% hint style="info" %}
- Προτεραιότητα: Προσαρμοσμένες παράμετροι > ενσωματωμένες παραμέτροι.
- Χρήση <kbd>όνομα:undefined</kbd> για εξαίρεση παραμέτρου.
- Μοναδικοί παράμετροι παρόχων βλ. την τεκμηρίωσή τους.
{% endhint %}
