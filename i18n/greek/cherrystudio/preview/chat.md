# Παρουσίαση Διαλόγου

## Βοηθοί και Θέματα

### Βοηθοί

Ο `βοηθός` αναφέρεται στις εξατομικευμένες ρυθμίσεις που εφαρμόζονται σε ένα επιλεγμένο μοντέλο για βελτιστοποιημένη χρήση, όπως προεπιλεγμένες εντολές και ρυθμίσεις παραμέτρων. Αυτές οι ρυθμίσεις βοηθούν το μοντέλο να ανταποκρίνεται καλύτερα στις αναμενόμενες ανάγκες.

Ο `προεπιλεγμένος βοηθός συστήματος` προσφέρει γενικές παραμέτρους (χωρίς προσθήκη εντολών). Μπορείτε είτε να τον χρησιμοποιήσετε απευθείας είτε να αναζητήσετε έτοιμες προεπιλογές στη [σελίδα πράκτορα](agents.md).

### Θέματα

Ο `βοηθός` λειτουργεί ως γονικό στοιχείο για τα `θέματα` (διαλόγους). Κάθε βοηθός μπορεί να δημιουργήσει πολλαπλά θέματα, όπου όλα τα θέματα μοιράζονται τις παραμέτρους και τις εντολές του αντίστοιχου βοηθού.

## Κουμπιά περιοχής διαλόγου

<figure><img src="../../.gitbook/assets/对话界面/对话框.png" alt="Παράθυρο διαλόγου"><figcaption></figcaption></figure>

![](../../.gitbook/assets/对话界面/新话题.png) `Νέο θέμα` Δημιουργεί νέο θέμα στον τρέχοντα βοηθό.

![](../../.gitbook/assets/对话界面/上传图片或文档.png) `Μεταφόρτωση εικόνας ή εγγράφου` Απαιτεί υποστήριξη από το μοντέλο για εικόνες. Τα έγγραφα μετατρέπονται αυτόματα σε κείμενο ως πλαίσιο.

![](../../.gitbook/assets/对话界面/网络搜索.png) `Διαδικτυακή αναζήτηση` Απαιτεί ρύθμιση παραμέτρων αναζήτησης. Τα αποτελέσματα χρησιμεύουν ως πλαίσιο για τα μοντέλα AI. Λεπτομέρειες: [Λειτουργία δικτύωσης](../../websearch/).

![](../../.gitbook/assets/对话界面/知识库.png) `Βάση γνώσης` Ενεργοποίηση βάσης γνώσης. Λεπτομέρειες: [Οδηγός βάσης γνώσης](../../knowledge-base/knowledge-base.md).

![](<../../.gitbook/assets/对话界面/MCP 服务器.png>) `Διακομιστής MCP` Ενεργοποίηση λειτουργικότητας διακομιστή MCP. Λεπτομέρειες: [Οδηγός MCP](../../advanced-basic/mcp/).

![](../../.gitbook/assets/对话界面/生成图片.png) `Δημιουργία εικόνας` Εμφανίζεται μόνο για μοντέλα με δυνατότητα δημιουργίας εικόνων (π.χ. Gemini). Απαιτεί χειροκίνητη ενεργοποίηση.

{% hint style="info" %}
Λόγω τεχνικών περιορισμών, απαιτείται χειροκίνητη ενεργοποίηση. Αυτό το κουμπί θα αφαιρεθεί μετά από μελλοντική βελτιστοποίηση.
{% endhint %}

![](../../.gitbook/assets/对话界面/选择模型.png) `Επιλογή μοντέλου` Αλλαγή μοντέλου για επόμενες απαντήσεις, με διατήρηση του πλαισίου.

![](../../.gitbook/assets/对话界面/快捷短语.png) `Προεπιλεγμένες φράσεις` Χρήση προκαθορισμένων φράσεων από ρυθμίσεις. Υποστηρίζει μεταβλητές.

![](../../.gitbook/assets/对话界面/清空消息.png) `Εκκαθάριση μηνυμάτων` Διαγραφή όλων των μηνυμάτων του τρέχοντος θέματος.

![](../../.gitbook/assets/对话界面/展开.png) `Ανάπτυξη` Μεγέθυνση περιοχής εισαγωγής για εκτενείς εισαγωγές.

![](../../.gitbook/assets/对话界面/清除上下文.png) `Εκκαθάριση πλαισίου` Περικοπή πλαισίου χωρίς διαγραφή μηνυμάτων. Το μοντέλο "ξεχνά" προηγούμενες απαντήσεις.

![](<../../.gitbook/assets/对话界面/预估 Token 数.png>) `Εκτίμηση αριθμού Token` Εμφάνιση εκτιμώμενου αριθμού Token:
- Τρέχον πλαίσιο
- Μέγιστο επιτρεπόμενο πλαίσιο (∞ = απεριόριστο)
- Τρέχον μήνυμα εισαγωγής
- Εκτιμώμενο πλήθος Token

{% hint style="info" %}
Για πειστικούς λόγους. Οι πραγματικοί αριθμοί Token διαφέρουν ανά μοντέλο. Ανατρέξτε σε δεδομένα παρόχου.
{% endhint %}

![](../../.gitbook/assets/对话界面/翻译.png) `Μετάφραση` Μετατροπή τρέχουσας εισαγωγής σε Αγγλικά.

## Ρυθμίσεις βοηθού

Επιλέξτε όνομα βοηθού → Δεξί κλικ → Επιλογή ρύθμισης

### Επεξεργασία βοηθού

{% hint style="info" %}
Ισχύει για όλα τα θέματα του βοηθού.
{% endhint %}

<figure><img src="../../.gitbook/assets/image (6) (1) (1).png" alt=""><figcaption></figcaption></figure>

#### Ρυθμίσεις εντολών

#### <mark style="color:blue;">**`Όνομα`**</mark>:

Προσαρμοσμένο αναγνωρίσιμο όνομα βοηθού.

#### <mark style="color:blue;">**`Εντολή`**</mark>:

Προσαρμογή προτροπής (σύμφωνα με τη δομή εντολών σελίδας πράκτορα).

#### Ρυθμίσεις μοντέλου

#### <mark style="color:blue;">**`Προεπιλεγμένο μοντέλο`**</mark>:

Καθορισμός προεπιλεγμένου μοντέλου για νέα θέματα (ανενεργό = χρήση [καθολικού προεπιλεγμένου](settings/default-models.md#default-assistant-model)).

{% hint style="info" %}
Υποστηρίζονται δύο προεπιλεγμένα: **καθολικό** και **βοηθού**. Η προτεραιότητα του βοηθού υπερβαίνει το καθολικό.
{% endhint %}

#### <mark style="color:blue;">**`Αυτόματη επαναφορά μοντέλου`**</mark>:

Επαναφορά προεπιλεγμένου μοντέλου σε νέα θέματα, αγνοώντας τις επιλογές του προηγούμενου θέματος.

> Παράδειγμα:<br>
> Προεπιλογή βοηθού: gpt-3.5-turbo<br>
> Αλλαγή σε gpt-4o κατά τη διάρκεια θέματος 1:<br>
> • **Ενεργό**: Θέμα 2 → gpt-3.5-turbo<br>
> • **Απενεργό**: Θέμα 2 → gpt-4o

#### <mark style="color:blue;">**`Θερμοκρασία (Temperature)`**</mark>:

Έλεγχος τυχαιότητας δημιουργικότητας (προεπιλογή: 0.7):
- **Χαμηλές τιμές (0-0.3)**: Ακριβής, προβλεπόμενες απαντήσεις
- **Μεσαίες τιμές (0.4-0.7)**: Ισορροπημένη δημιουργικότητα/συνέπεια (συνιστάται για διάλογο)
- **Υψηλές τιμές (0.8-1.0)**: Αυξημένη δημιουργικότητα (πιθανή απώλεια συνέπειας)

#### <mark style="color:blue;">**`Top P (πυρηνική δειγματοληψία)`**</mark>:

Προεπιλογή: 1. Μικρότερες τιμές αυξάνουν τον έλεγχο, μεγαλύτερες αυξάνουν την ποικιλομορφία:
- **Μικρές τιμές (0.1-0.3)**: Συντηρητικό, εστιασμένο (κατάλληλο για τεχνικό περιεχόμενο)
- **Μεσαίες τιμές (0.4-0.6)**: Ισορροπημένη ποικιλομορφία (γενική χρήση)
- **Μεγάλες τιμές (0.7-1.0)**: Αυξημένη ποικιλομορφία (δημιουργικές εργασίες)
