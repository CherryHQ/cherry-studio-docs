## Βοηθοί και Θέματα

### Βοηθός

`Βοηθός` αναφέρεται σε προσωπικοποιημένες ρυθμίσεις για το επιλεγμένο μοντέλο, όπως προεπιλεγμένες προτροπές και παράμετροι. Μέσω αυτών των ρυθμίσεων, το επιλεγμένο μοντέλο λειτουργεί πιο σύμφωνα με τις προσδοκίες σας.

`Προεπιλεγμένος βοηθός συστήματος` διαθέτει γενικές παραμέτρους (χωρίς προτροπές), μπορείτε να τον χρησιμοποιήσετε απευθείας ή να βρείτε τις απαιτούμενες προεπιλογές στη [σελίδα πρακτόρων](agents.md).

### Θέμα

`Βοηθός` είναι ο γονικός σύνολος των `θεμάτων`. Μπορείτε να δημιουργήσετε πολλά θέματα (δηλαδή συνομιλίες) υπό έναν βοηθό, και όλα τα `θέματα` μοιράζονται τις παραμέτρους και τις προεπιλεγμένες προτροπές (prompt) του `βοηθού`.

## Ρυθμίσεις Βοηθού

Στη σελίδα βοηθών, επιλέξτε το <mark style="background-color:yellow;">όνομα βοηθού</mark>→επιλέξτε τις αντίστοιχες ρυθμίσεις από το <mark style="background-color:yellow;">μενού δεξιού κλικ</mark>

### Επεξεργασία Βοηθού

{% hint style="info" %}
Οι ρυθμίσεις βοηθού ισχύουν για όλα τα θέματα υπό αυτόν τον βοηθό.
{% endhint %}

<figure><img src="../../.gitbook/assets/image (6) (1) (1).png" alt=""><figcaption></figcaption></figure>

#### Ρυθμίσεις προτροπών

#### <mark style="color:blue;">**`Όνομα`**</mark>：

Προσαρμοσμένο όνομα για ευκολότερη αναγνώριση.

#### <mark style="color:blue;">**`Προτροπή`**</mark>：

Δηλαδή προτροπή (prompt). Μπορείτε να ακολουθήσετε την τεχνική των προτροπών στη σελίδα πρακτόρων για την επεξεργασία.

#### Ρυθμίσεις μοντέλου

#### <mark style="color:blue;">**`Προεπιλεγμένο μοντέλο`**</mark>：

Μπορείτε να ορίσετε ένα προεπιλεγμένο μοντέλο για αυτόν τον βοηθό. Όταν προστίθεται από τη σελίδα πρακτόρων ή αντιγράφεται ο βοηθός, το αρχικό μοντέλο είναι αυτό. Αν δεν οριστεί, το αρχικό μοντέλο είναι το παγκόσμιο προεπιλεγμένο (δηλ. [Προεπιλεγμένο μοντέλο βοηθού](settings/default-models.md#mo-ren-zhu-shou-mo-xing)).

{% hint style="info" %}
Το προεπιλεγμένο μοντέλο βοηθού έχει δύο τύπους: ένας είναι το [Παγκόσμιο προεπιλεγμένο μοντέλο συνομιλίας](settings/default-models.md#mo-ren-zhu-shou-mo-xing), ο άλλος είναι το προεπιλεγμένο μοντέλο βοηθού. Το προεπιλεγμένο μοντέλο βοηθού έχει υψηλότερη προτεραιότητα. Όταν δεν οριστεί προεπιλεγμένο μοντέλο βοηθού, το προεπιλεγμένο μοντέλο βοηθού = Παγκόσμιο προεπιλεγμένο μοντέλο συνομιλίας.
{% endhint %}

#### <mark style="color:blue;">**`Αυτόματη επαναφορά μοντέλου`**</mark>：

Όταν ενεργοποιημένο - Αν κατά τη χρήση του θέματος αλλάξετε μοντέλο, η δημιουργία νέου θέματος θα επαναφέρει το μοντέλο στο προεπιλεγμένο του βοηθού. Όταν είναι απενεργοποιημένο, το μοντέλο του νέου θέματος ακολουθεί το μοντέλο του προηγούμενου θέματος.

> Π.χ.: Αν το προεπιλεγμένο μοντέλο βοηθού είναι gpt-3.5-turbo, δημιουργώ το θέμα 1 υπό τον βοηθό, και κατά τη συνομιλία αλλάζω στο gpt-4o:
>
> Αν είναι ενεργοποιημένη η αυτόματη επαναφορά: Κατά τη δημιουργία του θέματος 2, το προεπιλεγμένο μοντέλο είναι το gpt-3.5-turbo.
>
> Αν είναι απενεργοποιημένη: Κατά τη δημιουργία του θέματος 2, το προεπιλεγμένο μοντέλο είναι το gpt-4o.

#### <mark style="color:blue;">**`Θερμοκρασία (Temperature)`**</mark> ：

Η παράμετρος θερμοκρασίας ελέγχει τον βαθμό τυχαιότητας και δημιουργικότητας (προεπιλογή 0.7). Συγκεκριμένα:

* Χαμηλή τιμή (0-0.3):
  * Πιο βέβαιες και εστιασμένες απαντήσεις
  * Κατάλληλο για δημιουργία κώδικα, ανάλυση δεδομένων
  * Τείνει να επιλέγει τις πιθανότερες λέξεις
* Μεσαία τιμή (0.4-0.7):
  * Ισορροπία δημιουργικότητας και συνοχής
  * Κατάλληλο για καθημερινές συνομιλίες, γενικές εργασίες
  * Προτείνεται για chatbots (περίπου 0.5)
* Υψηλή τιμή (0.8-1.0):
  * Πιο δημιουργικές και ποικίλες απαντήσεις
  * Κατάλληλο για δημιουργική γραφή, brainstorming
  * Μπορεί να μειώσει τη συνοχή του κειμένου

#### <mark style="color:blue;">**`Top P (Πυρηνική δειγματοληψία)`**</mark>：

Προεπιλογή 1. Όσο μικρότερη η τιμή, τόσο πιο μονότονες και κατανοητές οι απαντήσεις. Όσο μεγαλύτερη, τόσο πιο ποικίλο το εύρος λεξιλογίου.

Η πυρηνική δειγματοληψία επηρεάζει την έξοδο μέσω του κατώφλιου πιθανότητας επιλογής λέξεων:

* Μικρή τιμή (0.1-0.3):
  * Θεωρεί μόνο τις πιο πιθανές λέξεις
  * Πιο συντηρητικές και ελεγχόμενες απαντήσεις
  * Κατάλληλο για σχόλια κώδικα, τεχνικά έγγραφα
* Μεσαία τιμή (0.4-0.6):
  * Ισορροπία ποικιλομορφίας και ακρίβειας
  * Κατάλληλο για γενικές συνομιλίες και εργασίες
* Μεγάλη τιμή (0.7-1.0):
  * Πιο εκτεταμένη επιλογή λέξεων
  * Παράγει πιο πλούσιο και ποικίλο περιεχόμενο
  * Κατάλληλο για δημιουργική γραφή

{% hint style="info" %}
- Οι παράμετροι μπορούν να χρησιμοποιηθούν ξεχωριστά ή σε συνδυασμό
- Επιλέξτε τις κατάλληλες τιμές ανάλογα με το είδος της εργασίας
- Συνιστάται δοκιμή για εύρεση του καλύτερου συνδυασμού
- Τα παραπάνω είναι για ενημέρωση. Οι συγκεκριμένες τιμές μπορεί να διαφέρουν ανά μοντέλο· ανατρέξτε στα έγγραφα του μοντέλου.
{% endhint %}

#### <mark style="color:blue;">**`Αριθμός πλαισίου αναφοράς (Context Window)`**</mark>

Αριθμός μηνυμάτων που διατηρούνται στο πλαίσιο αναφοράς. Πιο μεγάλη τιμή = πιο μεγάλο πλαίσιο, περισσότερα τοκέν:

* 5-10: Κατάλληλο για απλές συνομιλίες
* >10: Πολύπλοκες εργασίες που απαιτούν μεγάλη μνήμη (π.χ.: δημιουργία μακροσκελών κειμένων βήμα-βήμα)
* > Σημείωση: Περισσότερα μηνύματα = περισσότερα τοκέν

#### <mark style="color:blue;">**`Ενεργοποίηση ορίου μήκους μηνύματος (MaxToken)`**</mark>

Μέγιστος [Τοκέν](https://docs.cherry-ai.com/question-contact/knowledge#shen-me-shi-tokens) για κάθε απάντηση. Το max token είναι κρίσιμη παράμετρος που επηρεάζει την ποιότητα και το μήκος των απαντήσεων.

> Π.χ.: Στο CherryStudio, κατά τη δοκιμή σύνδεσης μοντέλου μετά την εισαγωγή του κλειδιού, αν χρειάζεστε μόνο να επιβεβαιώσετε τη σωστή απάντηση, ρυθμίστε το MaxToken σε 1.

Το περισσότερο μοντέλα έχουν όριο 32k τοκέν, αλλά υπάρχουν και 64k ή περισσότερα· ανατρέξτε στη σελίδα παρουσίασης του μοντέλου.

Η ρύθμιση εξαρτάται από τις ανάγκες σας. Παρακάτω προτάσεις:

{% hint style="success" %}
Προτάσεις:

* Απλή συνομιλία: 500-800
* Δημιουργία σύντομων κειμένων: 800-2000
* Δημιουργία κώδικα: 2000-3600
* Δημιουργία μακροσκελών κειμένων: 4000+ (απαιτεί υποστήριξη από το μοντέλο)
{% endhint %}

{% hint style="warning" %}
Συνήθως η απάντηση περιορίζεται στα MaxToken, αλλά μπορεί να υπάρξει αποκοπή (σε μακροσκελείς κώδικες) ή ατελής έκφραση. Προσαρμόζεται ανάλογα με την περίσταση.
{% endhint %}

#### <mark style="color:blue;">**`Ροϊκή έξοδος (Stream)`**</mark>

Η ροϊκή έξοδος επιτρέπει τη μετάδοση και επεξεργασία δεδομένων ως ροή, αντί για ολοκληρωμένη μετάδοση. Αυτό βελτιώνει την πραγματική απόδοση.

Σε περιβάλλοντα όπως το CherryStudio, αυτό σημαίνει εφέ πληκτρολόγησης.

Όταν απενεργοποιημένο (μη-ροϊκό): Η απάντηση εμφανίζεται ολόκληρη μετά τη δημιουργία (όπως σε WhatsApp).

Όταν ενεργοποιημένο: Εμφανίζεται χαρακτήρα-χαρακτήρα, λες ότι το μοντέλο στέλνει κάθε χαρακτήρα αμέσως.

{% hint style="info" %}
Για μοντέλα που δεν υποστηρίζουν ροϊκή έξοδο (π.χ. **αρχικά** το o1-mini), πρέπει να απενεργοποιήσετε αυτή την επιλογή.
{% endhint %}

#### <mark style="color:blue;">**`Προσαρμοσμένες παράμετροι`**</mark>

Προσθήκη επιπλέον παραμέτρων στο σώμα του αιτήματος (π.χ. `presence_penalty`). Συνήθως δεν χρειάζεται για τους περισσότερους χρήστες.

> Παράμετροι όπως to-p, maxtokens, stream είναι μέρος αυτών των παραμέτρων.

Μορφή: Όνομα παραμέτρου—Τύπος (κείμενο, αριθμός κ.λπ.)—Τιμή, αναφορά [εδώ](https://openai.apifox.cn/doc-3222739)

{% hint style="info" %}
Οι πάροχοι μοντέλων έχουν μοναδικές παραμέτρους. Ανατρέξτε στα έγγραφά τους.
{% endhint %}

{% hint style="info" %}
* Οι προσαρμοσμένες παράμετροι έχουν υψηλότερη προτεραιότητα από τις ενσωματωμένες. Αν υπάρχει επικάλυψη, οι προσαρμοσμένες παράμετροι έχουν προτεραιότητα.

> Π.χ.: Αν ορίσετε το `model` ως `gpt-4o` στις προσαρμοσμένες παραμέτρους, το μοντέλο θα είναι πάντα `gpt-4o`.

* Χρησιμοποιήστε <kbd>Όνομα παραμέτρου:undefined</kbd> για να αποκλείσετε παράμετρο.
{% endhint %}
