## Βοηθοί και Θέματα

### Βοηθός

Ο `Βοηθός` αναφέρεται στις εξατομικευμένες ρυθμίσεις που εφαρμόζονται στο επιλεγμένο μοντέλο, όπως προκαθορισμένες προτροπές και παράμετροι. Αυτές οι ρυθμίσεις βοηθούν το μοντέλο να ανταποκρίνεται καλύτερα στις αναμενόμενες εργασίες σας.

Ο `Προεπιλεγμένος Βοηθός Συστήματος` περιλαμβάνει γενικές παραμέτρους (χωρίς προκαθορισμένη προτροπή). Μπορείτε είτε να τον χρησιμοποιήσετε απευθείας είτε να μεταβείτε στη [σελίδα Ενσώματων Πρακτόρων](agents.md) για να βρείτε τις ρυθμίσεις που ανταποκρίνονται στις ανάγκες σας.

### Θέμα

Ο `Βοηθός` είναι το γονικό σύνολο των `θεμάτων`. Κάτω από έναν μεμονωμένο βοηθό, μπορείτε να δημιουργήσετε πολλαπλά θέματα (συνομιλίες). Όλα τα `θέματα` μοιράζονται τις ρυθμίσεις του βοηθού, όπως παραμέτρους και προκαθορισμένες προτροπές (prompt).

## Κουμπιά στο Παράθυρο Συνομιλίας

<figure><img src="../../.gitbook/assets/对话界面/对话框.png" alt=""><figcaption></figcaption></figure>

![](../../.gitbook/assets/对话界面/新话题.png) `Νέο Θέμα` Δημιουργεί ένα νέο θέμα στον τρέχοντα βοηθό.

![](../../.gitbook/assets/对话界面/上传图片或文档.png) `Ανέβασμα Εικόνας ή Εγγράφου` Το ανέβασμα εικόνων απαιτεί υποστήριξη από το μοντέλο. Τα ανεβασμένα έγγραφα μετατρέπονται αυτόματα σε κείμενο και παρέχονται ως πλαίσιο.

![](../../.gitbook/assets/对话界面/网络搜索.png) `Διαδικτυακή Αναζήτηση` Απαιτεί διαμόρφωση σχετικών πληροφοριών στις ρυθμίσεις. Τα αποτελέσματα επιστρέφονται ως πλαίσιο στο μοντέλο. Δείτε [Λειτουργία Διαδικτύου](../../websearch/).

![](../../.gitbook/assets/对话界面/知识库.png) `Βάση Γνώσεων` Ενεργοποιεί τη βάση γνώσεων. Δείτε [Οδηγός Βάσης Γνώσεων](../../knowledge-base/knowledge-base.md).

![](<../../.gitbook/assets/对话界面/MCP 服务器.png>) `Διακομιστής MCP` Ενεργοποιεί τη λειτουργία διακομιστή MCP. Δείτε [Οδηγός Χρήσης MCP](../../advanced-basic/mcp/).

![](../../.gitbook/assets/对话界面/生成图片.png) `Δημιουργία Εικόνας` Από προεπιλογή δεν εμφανίζεται. Για μοντέλα που υποστηρίζουν δημιουργία εικόνων (π.χ. Gemini), πρέπει να ενεργοποιηθεί χειροκίνητα.

{% hint style="info" %}
Λόγω τεχνικών λόγων, πρέπει να ενεργοποιήσετε χειροκίνητα το κουμπί για να δημιουργήσετε εικόνες. Αυτό το κουμπί θα αφαιρεθεί μετά τη βελτιστοποίηση αυτής της λειτουργίας.
{% endhint %}

![](../../.gitbook/assets/对话界面/选择模型.png) `Επιλογή Μοντέλου` Αλλάζει το μοντέλο για τις επόμενες συνομιλίες, διατηρώντας το πλαίσιο.

![](../../.gitbook/assets/对话界面/快捷短语.png) `Γρήγορες Φράσεις` Απαιτεί προκαθορισμένες κοινές φράσεις στις ρυθμίσεις. Υποστηρίζει μεταβλητές.

![](../../.gitbook/assets/对话界面/清空消息.png) `Εκκαθάριση Μηνυμάτων` Διαγράφει όλα τα περιεχόμενα του τρέχοντος θέματος.

![](../../.gitbook/assets/对话界面/展开.png) `Ανάπτυξη` Μεγεθύνει το παράθυρο συνομιλίας για εισαγωγή μεγάλων κειμένων.

![](../../.gitbook/assets/对话界面/清除上下文.png) `Εκκαθάριση Πλαισίου` Περικόπτει το πλαίσιο χωρίς διαγραφή περιεχομένων. Το μοντέλο "ξεχνά" τις προηγούμενες συνομιλίες.

![](<../../.gitbook/assets/对话界面/预估 Token 数.png>) `Εκτίμηση Αριθμού Token` Εμφανίζει: `τρέχον πλαίσιο`, `μέγιστο πλαίσιο` (∞ = απεριόριστο), `αριθμός λέξεων στο τρέχον μήνυμα`, `εκτιμώμενος αριθμός token`.

{% hint style="info" %}
Αυτή η λειτουργία παρέχει μόνο εκτιμήσεις. Ο πραγματικός αριθμός token διαφέρει ανά μοντέλο. Ανατρέξτε στα δεδομένα του παρόχου μοντέλου.
{% endhint %}

![](../../.gitbook/assets/对话界面/翻译.png) `Μετάφραση` Μεταφράζει το περιεχόμενο του τρέχοντος πλαισίου στα Αγγλικά.

### Ρυθμίσεις Μοντέλου

Συγχρονίζονται με τις παραμέτρους `Ρυθμίσεις Μοντέλου` στις ρυθμίσεις βοηθού. Δείτε [Επεξεργασία Βοηθού](chat.md#bian-ji-zhu-shou).

{% hint style="info" %}
Στις ρυθμίσεις συνομιλίας, μόνο οι ρυθμίσεις μοντέλου ισχύουν για τον τρέχοντα βοηθό. Οι υπόλοιπες ρυθμίσεις ισχύουν καθολικά (π.χ. ο ορισμός υφής μηνυμάτων ως φυσαλίδας).
{% endhint %}

#### <mark style="color:blue;">**`Αναδιπλούμενα Τμήματα Κώδικα`**</mark>:

Όταν είναι ενεργό, τα μεγάλα τμήματα κώδικα αναδιπλώνονται αυτόματα.

Όταν είναι ενεργό, η διαδικασία σκέψης των μοντέλων που την υποστηρίζουν πτυχώνεται αυτόματα μετά την ολοκλήρωσή της.

#### <mark style="color:blue;">**`Υφή Μηνυμάτων`**</mark>:

Αλλάζει τη διασύνδεση συνομιλίας σε στυλ φυσαλίδας ή λίστας.

Αλλάζει το στυλ εμφάνισης για τμήματα κώδικα.

#### <mark style="color:blue;">**`Μηχανισμός Μαθηματικών Τύπων`**</mark>:

* Το KaTeX αποδίδει γρηγορότερα λόγω βελτιστοποίησης απόδοσης.
* Το MathJax αποδίδει πιο αργά αλλά υποστηρίζει περισσότερα μαθηματικά σύμβολα και εντολές.

#### <mark style="color:blue;">**`Μέγεθος Γραμματοσειράς Μηνυμάτων`**</mark>:

Προσαρμόζει το μέγεθος γραμματοσειράς στη διασύνδεση συνομιλίας.

## Ρυθμίσεις Βοηθού

Επιλέξτε το <mark style="background-color:yellow;">όνομα βοηθού</mark> στη διασύνδεση βοηθών → Επιλέξτε τις αντίστοιχες ρυθμίσεις στο <mark style="background-color:yellow;">μενού δεξιού κλικ</mark>

### Επεξεργασία Βοηθού

{% hint style="info" %}
Οι ρυθμίσεις βοηθού ισχύουν για όλα τα θέματα κάτω από αυτόν τον βοηθό.
{% endhint %}

<figure><img src="../../.gitbook/assets/image (6) (1) (1).png" alt=""><figcaption></figcaption></figure>

#### Ρυθμίσεις Προτροπών

#### <mark style="color:blue;">**`Όνομα`**</mark>:

Προσαρμοσμένο όνομα για εύκολη αναγνώριση.

#### <mark style="color:blue;">**`Προτροπή`**</mark>:

Prompt. Μπορείτε να δείτε παραδείγματα στη σελίδα Ενσώματων Πρακτόρων.

#### Ρυθμίσεις Μοντέλου

#### <mark style="color:blue;">**`Προεπιλεγμένο Μοντέλο`**</mark>:

Μπορείτε να ορίσετε ένα σταθερό προεπιλεγμένο μοντέλο για τον βοηθό. Εάν δεν οριστεί, χρησιμοποιείται το καθολικό προεπιλεγμένο ([Προεπιλεγμένο Μοντέλο Βοηθού](settings/default-models.md#mo-ren-zhu-shou-mo-xing)).

{% hint style="info" %}
Υπάρχουν δύο τύποι: καθολικό προεπιλεγμένο μοντέλο συνομιλίας και προεπιλεγμένο μοντέλο βοηθού. Το προεπιλεγμένο μοντέλο βοηθού έχει προτεραιότητα έναντι του καθολικού. Αν δεν οριστεί, το προεπιλεγμένο μοντέλο βοηθού = καθολικό προεπιλεγμένο μοντέλο συνομιλίας.
{% endhint %}

#### <mark style="color:blue;">**`Αυτόματη Επαναφορά Μοντέλου`**</mark>:

Ενεργό → Όταν αλλάζετε μοντέλο σε ένα θέμα, τα νέα θέματα επαναφέρουν το προεπιλεγμένο μοντέλο του βοηθού. Ανενεργό → Τα νέα θέματα χρησιμοποιούν το τελευταίο μοντέλο του προηγούμενου θέματος.
