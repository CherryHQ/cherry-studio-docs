# Λειτουργίες Φωνής

```
Οδηγός χρήσης για τις λειτουργίες φωνής στο Cherry Studio

1. Επισκόπηση λειτουργιών φωνής

Το Cherry Studio παρέχει τρεις βασικές λειτουργίες φωνής: TTS (μετατροπή κειμένου σε ομιλία), ASR (αναγνώριση ομιλίας) και τηλεφωνική κλήση. Αυτές οι δυνατότητες σας επιτρέπουν να επικοινωνείτε με την ΤΝ με φυσικό τρόπο μέσω φωνής, βελτιώνοντας την εμπειρία χρήσης.

- TTS (μετατροπή κειμένου σε ομιλία): Μετατρέπει την απάντηση της ΤΝ σε ομιλία
- ASR (αναγνώριση ομιλίας): Μετατρέπει την ομιλία σας σε κείμενο
- Τηλεφωνική κλήση: Συνδυάζει TTS και ASR για εμπειρία φωνητικής συνομιλίας παρόμοια με το ChatGPT

2. Λειτουργία TTS (μετατροπή κειμένου σε ομιλία)

1. Υποστηριζόμενοι τύποι υπηρεσιών

Το Cherry Studio υποστηρίζει τέσσερις τύπους υπηρεσιών TTS:

- OpenAI: Χρησιμοποιεί το TTS API της OpenAI, απαιτεί κλειδί API
- TTS προγράμματος περιήγησης: Χρησιμοποιεί την ενσωματωμένη λειτουργία σύνθεσης ομιλίας του προγράμματος περιήγησης, δωρεάν χωρίς ρύθμιση
- Siliconflow: Χρησιμοποιεί την υπηρεσία TTS της Siliconflow, απαιτεί κλειδί API
- Δωρεάν διαδικτυακό TTS: Χρησιμοποιεί δωρεάν διαδικτυακή υπηρεσία TTS, χωρίς απαίτηση κλειδιού API

2. Μέθοδος ρύθμισης

1) Μεταβείτε στη σελίδα ρυθμίσεων, επιλέξτε την καρτέλα "Λειτουργίες φωνής"
2) Στην υποκαρτέλα "TTS":
   - Ενεργοποιήστε τη λειτουργία TTS (ενεργοποίηση διακόπτη)
   - Επιλέξτε τον τύπο υπηρεσίας TTS
   - Ανάλογα με τον επιλεγμένο τύπο υπηρεσίας, ρυθμίστε τις αντίστοιχες παραμέτρους:
     - OpenAI: Συμπληρώστε κλειδί API, διεύθυνση API, επιλέξτε ηχητικό ύφος και μοντέλο
     - TTS προγράμματος περιήγησης: Επιλέξτε ηχητικό ύφος
     - Siliconflow: Συμπληρώστε κλειδί API, διεύθυνση API, επιλέξτε ηχητικό ύφος, μοντέλο, μορφή απόκρισης και ταχύτητα ομιλίας
     - Δωρεάν διαδικτυακό TTS: Επιλέξτε ηχητικό ύφος και μορφή εξόδου
3) Ρύθμιση επιλογών φιλτραρίσματος TTS (προαιρετικό):
   - Φιλτράρισμα διαδικασιών σκέψης
   - Φιλτράρισμα σημάτων Markdown
   - Φιλτράρισμα μπλοκ κώδικα
4) Ορισμός εμφάνισης γραμμής προόδου TTS
5) Πατήστε το κουμπί "Τεστ TTS" για δοκιμή ρυθμίσεων

3. Μέθοδοι χρήσης

- Με την ενεργοποίηση του TTS, οι απαντήσεις της ΤΝ μετατρέπονται αυτόματα σε ομιλία
- Στο περιβάλλον συνομιλίας, κάθε απάντηση ΤΝ εμφανίζει κουμπί αναπαραγωγής TTS
- Πατήστε το κουμπί αναπαραγωγής για αναπαραγωγή/παύση ομιλίας
- Εάν η γραμμή προόδου TTS είναι ενεργοποιημένη, θα εμφανίζεται κάτω από το κείμενο
- Μεγάλα κείμενα διαχωρίζονται αυτόματα σε τμήματα και αναπαράγονται συνεχόμενα

3. Λειτουργία ASR (αναγνώριση ομιλίας)

1. Υποστηριζόμενοι τύποι υπηρεσιών

Το Cherry Studio υποστηρίζει τρεις τύπους υπηρεσιών ASR:

- OpenAI: Χρησιμοποιεί το μοντέλο Whisper της OpenAI, απαιτεί κλειδί API
- Πρόγραμμα περιήγησης: Χρησιμοποιεί την ενσωματωμένη λειτουργία αναγνώρισης ομιλίας του προγράμματος περιήγησης, δωρεάν χωρίς ρύθμιση
- Τοπικός διακομιστής: Σύνδεση με τοπικό διακομιστή WebSocket για αναγνώριση ομιλίας

2. Μέθοδος ρύθμισης

1) Μεταβείτε στη σελίδα ρυθμίσεων, επιλέξτε την καρτέλα "Λειτουργίες φωνής"
2) Στην υποκαρτέλα "ASR":
   - Ενεργοποιήστε τη λειτουργία ASR (ενεργοποίηση διακόπτη)
   - Επιλέξτε τον τύπο υπηρεσίας ASR
   - Ανάλογα με τον επιλεγμένο τύπο υπηρεσίας, ρυθμίστε τις αντίστοιχες παραμέτρους:
     - OpenAI: Συμπληρώστε κλειδί API, διεύθυνση API, επιλέξτε μοντέλο
     - Πρόγραμμα περιήγησης: Χωρίς επιπλέον ρύθμιση
     - Τοπικός διακομιστής: Μπορείτε να ορίσετε αυτόματη εκκίνηση διακομιστή ASR κατά την εκκίνηση της εφαρμογής
   - Επιλέξτε γλώσσα αναγνώρισης ομιλίας (προεπιλογή: κινέζικα)
3) Πατήστε το κουμπί "Τεστ ASR" για δοκιμή ρυθμίσεων

3. Μέθοδοι χρήσης

- Με την ενεργοποίηση του ASR, εμφανίζεται κουμπί αναγνώρισης ομιλίας δίπλα στο πλαίσιο εισαγωγής
- Πατήστε το κουμπί αναγνώρισης ομιλίας για έναρξη εγγραφής
- Μετά την ομιλία, η φωνή μετατρέπεται σε κείμενο και εισάγεται στο πλαίσιο
- Πατήστε ξανά το κουμπί για λήξη εγγραφής
- Υποστηρίζεται συνεχής αναγνώριση πολλαπλών φράσεων με αθροιστική λειτουργία

4. Λειτουργία τηλεφωνικής κλήσης

1. Χαρακτηριστικά λειτουργίας

- Συνδυασμός TTS και ASR για εμπειρία φωνητικής συνομιλίας παρόμοια με ChatGPT
- Διεπαφή με κινούμενο παράθυρο
- Υποστήριξη λειτουργίας με συνεχή πίεση για ομιλία
- Προσαρμογή συντόμευσης πληκτρολογίου
- Δυνατότητα αναδίπλωσης παραθύρου
- Επιλογή ειδικού μοντέλου τηλεφωνικής κλήσης
- Προσαρμογή προτροπών

2. Μέθοδος ρύθμισης

1) Μεταβείτε στη σελίδα ρυθμίσεων, επιλέξτε την καρτέλα "Λειτουργίες φωνής"
2) Στην υποκαρτέλα "Λειτουργία κλήσης":
   - Ενεργοποιήστε τη λειτουργία τηλεφωνικής κλήσης (ενεργοποίηση διακόπτη)
   - Πατήστε "Επιλογή μοντέλου" για επιλογή μοντέλου ΤΝ για τηλεφωνική κλήση
   - Προσαρμόστε προτροπές για τηλεφωνική κλήση στο πλαίσιο κειμένου (προαιρετικό)
   - Πατήστε "Αποθήκευση" για αποθήκευση προτροπών ή "Επαναφορά" για προεπιλεγμένες προτροπές

3. Μέθοδοι χρήσης

1) Στο περιβάλλον συνομιλίας, πατήστε το κουμπί τηλεφωνικής κλήσης (εικονίδιο τηλεφώνου) δεξιά του πλαισίου εισαγωγής
2) Θα ανοίξει το παράθυρο τηλεφωνικής κλήσης και θα αναπαραχθεί φωνητικό μήνυμα υποδοχής
3) Κρατήστε πατημένο το κουμπί "Κρατήστε πατημένο για ομιλία" για έναρξη εγγραφής (ή χρησιμοποιήστε τη ρυθμισμένη συντόμευση)
4) Αφήστε το κουμπί για λήξη εγγραφής και αποστολή στην ΤΝ
5) Η ΤΝ δημιουργεί απάντηση και την αναπαράγει μέσω TTS
6) Χρήση κουμπιών ελέγχου παραθύρου:
   - Κουμπί σίγασης: Ελέγχος εξόδου TTS
   - Κουμπί παύσης/συνέχειας: Παύση ή συνέχιση συνομιλίας
   - Κουμπί ρυθμίσεων: Διαμόρφωση συντόμευσης
   - Κουμπί αναδίπλωσης: Αναδίπλωση παραθύρου, διατήρηση μόνο της γραμμής ομιλίας
7) Πατήστε το κουμπί κλεισίματος για τερματισμό κλήσης

4. Ρύθμιση συντόμευσης πληκτρολογίου

1) Στο παράθυρο τηλεφωνικής κλήσης, πατήστε το κουμπί ρυθμίσεων
2) Στο πλαίσιο ρυθμίσεων, πατήστε το κουμπί συντόμευσης
3) Πατήστε το επιθυμητό πλήκτρο (π.χ. space, shift)
4) Πατήστε "Αποθήκευση"
5) Κατά τη χρήση, κρατήστε πατημένη τη συντόμευση για εγγραφή, αφήστε για αποστολή

5. Συχνά προβλήματα και λύσεις

1. Προβλήματα TTS

- Πρόβλημα: Αδυναμία αναπαραγωγής ήχου TTS
  Λύση: Ελέγξτε ενεργοποίηση TTS, σωστό τύπο υπηρεσίας και απαραίτητες παραμέτρους

- Πρόβλημα: Χαμηλή ποιότητα αναπαραγωγής TTS
  Λύση: Δοκιμάστε διαφορετικό τύπο υπηρεσίας TTS ή ηχητικό ύφος

- Πρόβλημα: Μήνυμα σφάλματος κατά την αναπαραγωγή TTS
  Λύση: Ελέγξτε ορθότητα κλειδιού API και σύνδεση δικτύου

2. Προβλήματα ASR

- Πρόβλημα: ASR δεν αναγνωρίζει ομιλία
  Λύση: Ελέγξτε ενεργοποίηση ASR, σωστό τύπο υπηρεσίας και απαραίτητες παραμέτρους

- Πρόβλημα: Χαμηλή ακρίβεια αναγνώρισης ASR
  Λύση: Δοκιμάστε διαφορετικό τύπο υπηρεσίας ASR ή βελτιστοποιήστε τη θέση/ένταση μικροφώνου

- Πρόβλημα: Αποτυχία σύνδεσης με διακομιστή ASR
  Λύση: Ελέγξτε λειτουργία τοπικού διακομιστή ή επανεκκινήστε την εφαρμογή

3. Προβλήματα τηλεφωνικής κλήσης

- Πρόβλημα: Αδυναμία ανοίγματος παραθύρου κλήσης
  Λύση: Ελέγξτε ενεργοποίηση τηλεφωνικής κλήσης και σωστές ρυθμίσεις TTS/ASR

- Πρόβλημα: Χωρίς απόκριση με συνεχή πίεση για ομιλία
  Λύση: Ελέγξτε δικαιώματα μικροφώνου ή επανεκκινήστε τη λειτουργία κλήσης

- Πρόβλημα: Χωρίς φωνητική έξοδο από ΤΝ
  Λύση: Ελέγξτε ενεργοποίηση TTS και απενεργοποίηση σίγασης

6. Προηγμένες ρυθμίσεις και προσαρμοσμένες επιλογές

1. Προηγμένες ρυθμίσεις TTS

- Επιλογές φιλτραρίσματος: Φιλτράρισμα διαδικασιών σκέψης, σημάτων Markdown και μπλοκ κώδικα για ομαλότερη αναπαραγωγή
- Εμφάνιση γραμμής προόδου: Ενεργοποίηση/απενεργοποίηση
- Προσαρμοσμένα ηχητικά ύφη και μοντέλα: Προσθήκη προσαρμοσμένων επιλογών

2. Προηγμένες ρυθμίσεις ASR

- Αυτόματη εκκίνηση διακομιστή: Ενεργοποίηση αυτόματης εκκίνησης διακομιστή ASR κατά την εκκίνηση εφαρμογής
- Επιλογή γλώσσας: Επιλογή διαφορετικών γλωσσών αναγνώρισης ομιλίας

3. Προηγμένες ρυθμίσεις τηλεφωνικής κλήσης

- Προσαρμοσμένες προτροπές: Προσαρμογή προτροπών για επικοινωνία ΤΝ σε λειτουργία κλήσης
- Επιλογή ειδικού μοντέλου: Επιλογή ξεχωριστού μοντέλου ΤΝ για τηλεφωνικές κλήσεις
- Προσαρμογή συντόμευσης: Ορισμός προσαρμοσμένης συντόμευσης για ελέγχω εγγραφής

7. Συμβουλές χρήσης

1. Επιλογή κατάλληλης υπηρεσίας TTS:
   - Για υψηλή ποιότητα ομιλίας: OpenAI ή Siliconflow
   - Χωρίς ανάγκη ρύθμισης API: TTS προγράμματος περιήγησης ή δωρεάν διαδικτυακό TTS

2. Επιλογή κατάλληλης υπηρεσίας ASR:
   - Για υψηλή ακρίβεια: OpenAI
   - Χωρίς ανάγκη ρύθμισης API: Ενσωματωμένη αναγνώριση προγράμματος περιήγησης

3. Βελτιστοποίηση εμπειρίας τηλεφωνικής κλήσης:
   - Χρήση ακουστικών αποφεύγει επαναληπτική κατα
