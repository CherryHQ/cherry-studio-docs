# macOS

1. Αρχικά, μεταβείτε στη σελίδα λήψης της επίσημης ιστοσελίδας και κάντε κλικ για να κατεβάσετε την έκδοση για Mac, ή κάντε κλικ απευθείας παρακάτω  

Παρακαλώ λάβετε υπόψη ότι πρέπει να κατεβάσετε **την έκδοση που αντιστοιχεί στο τσιπ του Mac σας**
{% hint style="info" %}
Εάν δεν γνωρίζετε ποια έκδοση τσιπ πρέπει να χρησιμοποιήσετε για τον Mac σας:

* Κάντε κλικ στο μενού  στην επάνω αριστερή γωνία της γραμμής μενού
* Στο αναπτυσσόμενο μενού, επιλέξτε "Σχετικά με αυτό το Mac"
* Στο αναδυόμενο παράθυρο, ελέγξτε τις πληροφορίες της επεξεργαστικής μονάδας

Εάν είναι τσιπ Intel, κατεβάστε το πακέτο εγκατάστασης έκδοσης Intel

Εάν είναι τσιπ Apple M\*, κατεβάστε το πακέτο εγκατάστασης για τσιπ Apple
{% endhint %}
