# Λειτουργία Σύνδεσης στο Διαδίκτυο


{% hint style="warning" %}
Αυτό το έγγραφο μεταφράστηκε από τα Κινεζικά με AI και δεν έχει ακόμη ελεγχθεί.
{% endhint %}




{% hint style="info" %}
Παραδείγματα περιπτώσεων που απαιτείται σύνδεση στο διαδίκτυο:

* Πληροφορίες με χρονική σημασία: π.χ. η τιμή των χρυσών συμβάσεων σήμερα/αυτή την εβδομάδα/πρόσφατα.
* Δυναμικά δεδομένα: π.χ. καιρός, συναλλαγματικές ισοτιμίες κ.λπ.
* Νέα γνώση: π.χ. νέα φαινόμενα, έννοιες, τεχνολογίες κ.λπ.
{% endhint %}

### 1. Πώς να ενεργοποιήσετε τη σύνδεση στο διαδίκτυο

Στο παράθυρο ερωτήσεων του Cherry Studio, κάντε κλικ στο εικονίδιο 【Μικρή Γη】 για να ενεργοποιήσετε τη σύνδεση.

<figure><img src="../.gitbook/assets/image (94).png" alt=""><figcaption><p>Κάντε κλικ στο εικονίδιο Γης - Ενεργοποίηση σύνδεσης</p></figcaption></figure>

<figure><img src="../.gitbook/assets/image (96).png" alt=""><figcaption><p>Ενδειξη - Λειτουργία σύνδεσης ενεργοποιημένη</p></figcaption></figure>

### 2. Σημαντική σημείωση: Δύο τύποι λειτουργίας σύνδεσης

#### Τύπος 1: Η ενσωματωμένη λειτουργία σύνδεσης των μεγάλων γλωσσικών μοντέλων

Σε αυτήν την περίπτωση, αφού ενεργοποιήσετε τη σύνδεση, μπορείτε να χρησιμοποιήσετε άμεσα τις υπηρεσίες διαδικτύου.

{% hint style="warning" %}
Μπορείτε να διαπιστώσετε γρήγορα αν ένα μοντέλο υποστηρίζει σύνδεση ελέγχοντας αν το όνομα του μοντέλου συνοδεύεται από το εικονίδιο μικρού χάρτη στη γραμμή ερωτήσεων.
{% endhint %}

<figure><img src="../.gitbook/assets/image (100).png" alt=""><figcaption></figcaption></figure>

Αυτή η μέθοδος σας επιτρέπει επίσης να διακρίνετε γρήγορα ποια μοντέλα υποστηρίζουν σύνδεση στη σελίδα διαχείρισης μοντέλων.

<figure><img src="../.gitbook/assets/image (101).png" alt=""><figcaption></figcaption></figure>

> <mark style="color:green;">**Πάροχοι μοντέλων με υποστήριξη σύνδεσης στο Cherry Studio**</mark>
>
> * <mark style="color:green;">Google Gemini</mark>
> * <mark style="color:green;">OpenRouter (όλα τα μοντέλα υποστηρίζουν σύνδεση)</mark>
> * <mark style="color:green;">Tencent Hunyuan</mark>
> * <mark style="color:green;">Zhipu AI</mark>
> * <mark style="color:green;">Alibaba Cloud Bailian κ.ά.</mark>

{% hint style="danger" %}
Σημαντική σημείωση:

Υπάρχει μια ειδική περίπτωση όπου ακόμη και αν το μοντέλο δεν έχει το εικονίδιο μικρού χάρτη, μπορεί να υποστηρίζει σύνδεση, όπως εξηγείται στο παρακάτω έγγραφο.
{% endhint %}

#### Τύπος 2: Χρήση της υπηρεσίας Tavily για σύνδεση

Όταν χρησιμοποιούμε ένα γλωσσικό μοντέλο χωρίς ενσωματωμένη υποστήριξη σύνδεσης (χωρίς το εικονίδιο μικρού χάρτη), αλλά χρειαζόμαστε πρόσβαση σε πραγματικού χρόνου πληροφορίες, τότε χρησιμοποιούμε την υπηρεσία αναζήτησης Tavily.

**Κατά την πρώτη χρήση της υπηρεσίας Tavily**, θα εμφανιστεί παράθυρο με οδηγίες ρύθμισης. Απλώς ακολουθήστε τις οδηγίες - είναι πολύ απλό!

<figure><img src="../.gitbook/assets/image (102).png" alt=""><figcaption><p>Παράθυρο - Κάντε κλικ: Μετάβαση στις ρυθμίσεις</p></figcaption></figure>

<figure><img src="../.gitbook/assets/image (104).png" alt=""><figcaption><p>Κάντε κλικ: Λήψη κλειδιού</p></figcaption></figure>

Αφού κάνετε κλικ στη "Λήψη κλειδιού", θα μεταφερθείτε αυτόματα στη σελίδα **εγγραφής/σύνδεσης του tavily**. Αφού εγγραφείτε και συνδεθείτε, δημιουργήστε ένα API κλειδί και αντιγράψτε το στο Cherry Studio.

{% hint style="danger" %}
Αν αντιμετωπίζετε προβλήματα με την εγγραφή, ανατρέξτε στον οδηγό εγγραφής tavily στον ίδιο κατάλογο.
{% endhint %}

**Έγγραφο αναφοράς για εγγραφή στο tavily:**

{% content-ref url="tavily.md" %}
[tavily.md](tavily.md)
{% endcontent-ref %}

Η παρακάτω οθόνη υποδεικνύει επιτυχή εγγραφή.

<figure><img src="../.gitbook/assets/image (105).png" alt=""><figcaption><p>Αντιγραφή κλειδιού</p></figcaption></figure>

<figure><img src="../.gitbook/assets/image (108).png" alt=""><figcaption><p>Επικόλληση κλειδιού - Έτοιμο!</p></figcaption></figure>
