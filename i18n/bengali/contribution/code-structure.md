# কোড স্ট্রাকচার

```yaml
...
├─ docs/ #ডকুমেন্টেশন ডিরেক্টরি
├─ resources/ #রিসোর্স ফাইল ডিরেক্টরি
├─ scripts/ #স্ক্রিপ্ট ফাইল ডিরেক্টরি
└─ src/ #প্রধান সোর্স কোড ডিরেক্টরি
    ├─main/ #মেইন প্রসেস কোড
    ├─preload/ #প্রিলোড স্ক্রিপ্ট ডিরেক্টরি
    └─renderer/ #রেন্ডারার প্রসেস কোড
        ├─src/ #রেন্ডারারের সোর্স কোড
            ├─assets/ #রিসোর্স ফাইল
                ├─fonts/ #ফন্ট ফাইল
                ├─images/ #প্রোফাইল পিকচার ইত্যাদি ইমেজ ফাইল
                └─styles/ #স্টাইল ফাইল
            ├─components/ #কম্পোনেন্টস
            ├─config/ #কনফিগারেশন ফাইল
            ├─context/ #কনটেক্সট
            ├─databases/ #ডাটাবেস সংক্রান্ত ফাইল
            ├─hooks/ #কাস্টম হুকস
            ├─i18n/ #আন্তর্জাতিকীকরণ ফাইল
            ├─pages/ #পেজ ফাইল
            ├─providers/ #সার্ভিস প্রোভাইডার সংক্রান্ত কনফিগারেশন
            ├─services/ #সার্ভিস ফাইল
            ├─store/ #স্টেট ম্যানেজমেন্ট ফাইল
            ├─types/ #টাইপস্ক্রিপ্ট টাইপ ডেফিনিশন ফাইল
            ├─utils/ #ইউটিলিটি ফাংশন
            ...
        ...
    ...
...
```
