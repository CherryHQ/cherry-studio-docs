# 語音功能

```
Cherry Studio 語音功能使用說明

一、語音功能概述

Cherry Studio 提供三大語音功能模組：TTS（文本轉語音）、ASR（語音識別）和語音通話。這些功能讓您能透過語音與AI進行自然交流，提升使用體驗。

- TTS（文本轉語音）：將AI回覆的文本轉換為語音輸出
- ASR（語音識別）：將您的語音轉換為文本輸入
- 語音通話：結合TTS和ASR，實現類似ChatGPT的語音對話體驗

二、TTS（文本轉語音）功能

1. 支援的服務類型

Cherry Studio 支援四種TTS服務類型：

- OpenAI：使用OpenAI的TTS API，需要API金鑰
- 瀏覽器TTS：使用瀏覽器內建的語音合成功能，免費無需配置
- 矽基流動：使用矽基流動（Siliconflow）的TTS服務，需要API金鑰
- 免費在線TTS：使用免費的在線TTS服務，無需API金鑰

2. 設定方法

1) 進入設定頁面，選擇「語音功能」選項卡
2) 在「TTS」子選項卡中：
   - 啟用TTS功能（打開開關）
   - 選擇TTS服務類型
   - 根據選擇的服務類型，配置相應參數：
     - OpenAI：填寫API金鑰、API地址、選擇音色和模型
     - 瀏覽器TTS：選擇音色
     - 矽基流動：填寫API金鑰、API地址、選擇音色、模型、響應格式和語速
     - 免費在線TTS：選擇音色和輸出格式
3) 配置TTS過濾選項（可選）：
   - 過濾思考過程
   - 過濾Markdown標記
   - 過濾代碼塊
4) 設定是否顯示TTS進度條
5) 點擊「測試TTS」按鈕測試配置是否正確

3. 使用方法

- 啟用TTS功能後，AI的回覆會自動轉換為語音輸出
- 在聊天界面中，每條AI回覆下方會顯示TTS播放按鈕
- 點擊播放按鈕可以播放/暫停語音
- 如果啟用了TTS進度條，會在文本下方顯示播放進度
- 長文本會自動分段合成並連續播放

三、ASR（語音識別）功能

1. 支援的服務類型

Cherry Studio 支援三種ASR服務類型：

- OpenAI：使用OpenAI的Whisper模型，需要API金鑰
- 瀏覽器：使用瀏覽器內建的語音識別功能，免費無需配置
- 本地伺服器：連接到本地WebSocket伺服器進行語音識別

2. 設定方法

1) 進入設定頁面，選擇「語音功能」選項卡
2) 在「ASR」子選項卡中：
   - 啟用ASR功能（打開開關）
   - 選擇ASR服務類型
   - 根據選擇的服務類型，配置相應參數：
     - OpenAI：填寫API金鑰、API地址、選擇模型
     - 瀏覽器：無需額外配置
     - 本地伺服器：可以設定是否在應用啟動時自動啟動ASR伺服器
   - 選擇語音識別語言（默認為中文）
3) 點擊「測試ASR」按鈕測試配置是否正確

3. 使用方法

- 啟用ASR功能後，在輸入框旁邊會顯示語音識別按鈕
- 點擊語音識別按鈕開始錄音
- 說話後，語音會被轉換為文本並填入輸入框
- 再次點擊按鈕結束錄音
- 語音識別支援連續識別多句話，採用累加模式

四、語音通話功能

1. 功能特點

- 結合TTS和ASR，實現類似ChatGPT的語音對話體驗
- 使用可拖動的懸浮窗口界面
- 支援長按說話模式
- 支援自訂快捷鍵
- 支援窗口摺疊
- 可以選擇專門的語音通話模型
- 支援自訂提示詞

2. 設定方法

1) 進入設定頁面，選擇「語音功能」選項卡
2) 在「通話功能」子選項卡中：
   - 啟用語音通話功能（打開開關）
   - 點擊「選擇模型」按鈕，選擇用於語音通話的AI模型
   - 在提示詞文字框中自訂語音通話提示詞（可選）
   - 點擊「儲存」按鈕儲存提示詞，或點擊「重置」按鈕恢復預設提示詞

3. 使用方法

1) 在聊天界面中，點擊輸入框右側的語音通話按鈕（電話圖標）
2) 語音通話窗口會打開，並播放歡迎語音
3) 長按「長按說話」按鈕開始錄音（或使用設定的快捷鍵）
4) 鬆開按鈕結束錄音並發送給AI處理
5) AI生成回覆並透過TTS播放
6) 使用窗口中的控制按鈕：
   - 靜音/取消靜音按鈕：控制TTS輸出
   - 暫停/繼續按鈕：暫停或繼續對話
   - 設定按鈕：配置快捷鍵
   - 摺疊按鈕：摺疊窗口，只保留長按說話那一行
7) 點擊關閉按鈕結束通話

4. 快捷鍵設定

1) 在語音通話窗口中，點擊設定按鈕
2) 在彈出的設定面板中，點擊快捷鍵按鈕
3) 按下您想要設定的鍵（如空格鍵、Shift鍵等）
4) 點擊「儲存」按鈕儲存設定
5) 使用時，按住設定的快捷鍵開始錄音，鬆開結束錄音並發送

五、常見問題和解決方案

1. TTS 相關問題

- 問題：TTS 無法播放聲音
  解決方案：檢查是否啟用了TTS功能，確保選擇了正確的服務類型和配置了必要的參數

- 問題：TTS 播放品質不佳
  解決方案：嘗試更換不同的TTS服務類型或音色

- 問題：TTS 播放時顯示錯誤信息
  解決方案：檢查API金鑰是否正確，網路連接是否正常

2. ASR 相關問題

- 問題：ASR 無法識別語音
  解決方案：檢查是否啟用了ASR功能，確保選擇了正確的服務類型和配置了必要的參數

- 問題：ASR 識別準確率低
  解決方案：嘗試更換不同的ASR服務類型，或調整麥克風位置和音量

- 問題：ASR 伺服器連接失敗
  解決方案：檢查本地伺服器是否正常運行，或嘗試重啟應用

3. 語音通話相關問題

- 問題：語音通話窗口無法打開
  解決方案：檢查是否啟用了語音通話功能，確保TTS和ASR功能配置正確

- 問題：長按說話沒有反應
  解決方案：檢查麥克風權限是否已授予，或嘗試重新啟動語音通話

- 問題：AI 回覆沒有語音輸出
  解決方案：檢查是否啟用了TTS功能，確保沒有靜音

六、進階設定和自訂選項

1. TTS 進階設定

- 過濾選項：可以選擇過濾思考過程、Markdown標記和代碼塊，使TTS播放更流暢
- 進度條顯示：可以選擇是否顯示TTS進度條
- 自訂音色和模型：可以添加自訂的音色和模型選項

2. ASR 進階設定

- 自動啟動伺服器：可以設定應用啟動時是否自動啟動ASR伺服器
- 語言選擇：可以選擇不同的語音識別語言

3. 語音通話進階設定

- 自訂提示詞：可以自訂語音通話提示詞，指導AI在語音通話模式下的回覆方式
- 專用模型選擇：可以為語音通話選擇專門的AI模型，與目前對話使用的模型分開
- 快捷鍵自訂：可以設定自訂快捷鍵來控制錄音

七、使用建議

1. 選擇合適的TTS服務：
   - 如果追求高品質語音，推薦使用OpenAI或矽基流動
   - 如果不想配置API，可以使用瀏覽器TTS或免費在線TTS

2. 選擇合適的ASR服務：
   - 如果追求高準確率，推薦使用OpenAI
   - 如果不想配置API，可以使用瀏覽器內建的語音識別

3. 優化語音通話體驗：
   - 使用耳機可以避免TTS輸出被ASR再次捕獲
   - 在安靜的環境中使用可以提高識別準確率
   - 使用自訂提示詞可以讓AI回覆更適合語音播放

4. 根據需求調整設定：
   - 如果主要使用文本交流，可以只啟用TTS功能
   - 如果主要使用語音輸入，可以只啟用ASR功能
   - 如果需要完整的語音對話體驗，啟用語音通話功能

希望這份使用說明能幫助您充分利用Cherry Studio的語音功能，享受更自然、便捷的AI互動體驗！
```
