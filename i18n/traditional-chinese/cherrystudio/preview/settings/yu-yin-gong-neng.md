# 語音功能

```
Cherry Studio 語音功能使用說明

一、語音功能概述

Cherry Studio 提供三大語音功能模組：TTS（文本轉語音）、ASR（語音辨識）和語音通話。這些功能讓您可以通過語音與 AI 進行自然交流，提升使用體驗。

- TTS（文本轉語音）：將 AI 回覆的文本轉換為語音輸出
- ASR（語音辨識）：將您的語音轉換為文本輸入
- 語音通話：結合 TTS 和 ASR，實現類似 ChatGPT 的語音對話體驗

二、TTS（文本轉語音）功能

1. 支援的服務類型

Cherry Studio 支援四種 TTS 服務類型：

- OpenAI：使用 OpenAI 的 TTS API，需要 API 密鑰
- 瀏覽器 TTS：使用瀏覽器內建的語音合成功能，免費無需配置
- 矽基流動：使用矽基流動（Siliconflow）的 TTS 服務，需要 API 密鑰
- 免費在線 TTS：使用免費的在線 TTS 服務，無需 API 密鑰

2. 設定方法

1) 進入設定頁面，選擇「語音功能」選項卡
2) 在「TTS」子選項卡中：
   - 啟用 TTS 功能（打開開關）
   - 選擇 TTS 服務類型
   - 根據選擇的服務類型，配置相應參數：
     - OpenAI：填寫 API 密鑰、API 地址、選擇音色和模型
     - 瀏覽器 TTS：選擇音色
     - 矽基流動：填寫 API 密鑰、API 地址、選擇音色、模型、回應格式和語速
     - 免費在線 TTS：選擇音色和輸出格式
3) 配置 TTS 過濾選項（可選）：
   - 過濾思考過程
   - 過濾 Markdown 標記
   - 過濾代碼塊
4) 設定是否顯示 TTS 進度條
5) 點擊「測試 TTS」按鈕測試配置是否正確

3. 使用方法

- 啟用 TTS 功能後，AI 的回覆會自動轉換為語音輸出
- 在聊天界面中，每條 AI 回覆下方會顯示 TTS 播放按鈕
- 點擊播放按鈕可以播放/暫停語音
- 如果啟用了 TTS 進度條，會在文本下方顯示播放進度
- 長文本會自動分段合成並連續播放

三、ASR（語音辨識）功能

1. 支援的服務類型

Cherry Studio 支援三種 ASR 服務類型：

- OpenAI：使用 OpenAI 的 Whisper 模型，需要 API 密鑰
- 瀏覽器：使用瀏覽器內建的語音辨識功能，免費無需配置
- 本地伺服器：連接到本地 WebSocket 伺服器進行語音辨識

2. 設定方法

1) 進入設定頁面，選擇「語音功能」選項卡
2) 在「ASR」子選項卡中：
   - 啟用 ASR 功能（打開開關）
   - 選擇 ASR 服務類型
   - 根據選擇的服務類型，配置相應參數：
     - OpenAI：填寫 API 密鑰、API 地址、選擇模型
     - 瀏覽器：無需額外配置
     - 本地伺服器：可以設定是否在應用啟動時自動啟動 ASR 伺服器
   - 選擇語音辨識語言（預設為中文）
3) 點擊「測試 ASR」按鈕測試配置是否正確

3. 使用方法

- 啟用 ASR 功能後，在輸入框旁邊會顯示語音辨識按鈕
- 點擊語音辨識按鈕開始錄音
- 說話後，語音會被轉換為文本並填入輸入框
- 再次點擊按鈕結束錄音
- 語音辨識支援連續辨識多句話，採用累加模式

四、語音通話功能

1. 功能特點

- 結合 TTS 和 ASR，實現類似 ChatGPT 的語音對話體驗
- 使用可拖動的懸浮視窗界面
- 支援長按說話模式
- 支援自訂快捷鍵
- 支援視窗折疊
- 可以選擇專門的語音通話模型
- 支援自訂提示詞

2. 設定方法

1) 進入設定頁面，選擇「語音功能」選項卡
2) 在「通話功能」子選項卡中：
   - 啟用語音通話功能（打開開關）
   - 點擊「選擇模型」按鈕，選擇用於語音通話的 AI 模型
   - 在提示詞文本框中自訂語音通話提示詞（可選）
   - 點擊「保存」按鈕保存提示詞，或點擊「重置」按鈕恢復預設提示詞

3. 使用方法

1) 在聊天界面中，點擊輸入框右側的語音通話按鈕（電話圖標）
2) 語音通話視窗會打開，並播放歡迎語音
3) 長按「長按說話」按鈕開始錄音（或使用設定的快捷鍵）
4) 鬆開按鈕結束錄音並發送給 AI 處理
5) AI 生成回覆並通過 TTS 播放
6) 使用視窗中的控制按鈕：
   - 靜音/取消靜音按鈕：控制 TTS 輸出
   - 暫停/繼續按鈕：暫停或繼續對話
   - 設定按鈕：配置快捷鍵
   - 折疊按鈕：折疊視窗，只保留長按說話那一行
7) 點擊關閉按鈕結束通話

4. 快捷鍵設定

1) 在語音通話視窗中，點擊設定按鈕
2) 在彈出的設定面板中，點擊快捷鍵按鈕
3) 按下您想要設定的鍵（如空白鍵、Shift 鍵等）
4) 點擊「保存」按鈕保存設定
5) 使用時，按住設定的快捷鍵開始錄音，鬆開結束錄音並發送

五、常見問題和解決方案

1. TTS 相關問題

- 問題：TTS 無法播放聲音
  解決方案：檢查是否啟用了 TTS 功能，確保選擇了正確的服務類型和配置了必要的參數

- 問題：TTS 播放質量不佳
  解決方案：嘗試更換不同的 TTS 服務類型或音色

- 問題：TTS 播放時顯示錯誤信息
  解決方案：檢查 API 密鑰是否正確，網絡連接是否正常

2. ASR 相關問題

- 問題：ASR 無法辨識語音
  解決方案：檢查是否啟用了 ASR 功能，確保選擇了正確的服務類型和配置了必要的參數

- 問題：ASR 辨識準確率低
  解決方案：嘗試更換不同的 ASR 服務類型，或調整麥克風位置和音量

- 問題：ASR 伺服器連接失敗
  解決方案：檢查本地伺服器是否正常運行，或嘗試重啟應用

3. 語音通話相關問題

- 問題：語音通話視窗無法打開
  解決方案：檢查是否啟用了語音通話功能，確保 TTS 和 ASR 功能配置正確

- 問題：長按說話沒有反應
  解決方案：檢查麥克風權限是否已授予，或嘗試重新啟動語音通話

- 問題：AI 回覆沒有語音輸出
  解決方案：檢查是否啟用了 TTS 功能，確保沒有靜音

六、高級設定和自訂選項

1. TTS 高級設定

- 過濾選項：可以選擇過濾思考過程、Markdown 標記和代碼塊，使 TTS 播放更流暢
- 進度條顯示：可以選擇是否顯示 TTS 進度條
- 自訂音色和模型：可以添加自訂的音色和模型選項

2. ASR 高級設定

- 自動啟動伺服器：可以設定應用啟動時是否自動啟動 ASR 伺服器
- 語言選擇：可以選擇不同的語音辨識語言

3. 語音通話高級設定

- 自訂提示詞：可以自訂語音通話提示詞，指導 AI 在語音通話模式下的回覆方式
- 專用模型選擇：可以為語音通話選擇專門的 AI 模型，與當前對話使用的模型分開
- 快捷鍵自訂：可以設定自訂快捷鍵來控制錄音

七、使用建議

1. 選擇合適的 TTS 服務：
   - 如果追求高質量語音，推薦使用 OpenAI 或矽基流動
   - 如果不想配置 API，可以使用瀏覽器 TTS 或免費在線 TTS

2. 選擇合適的 ASR 服務：
   - 如果追求高準確率，推薦使用 OpenAI
   - 如果不想配置 API，可以使用瀏覽器內建的語音辨識

3. 優化語音通話體驗：
   - 使用耳機可以避免 TTS 輸出被 ASR 再次捕獲
   - 在安靜的環境中使用可以提高辨識準確率
   - 使用自訂提示詞可以讓 AI 回覆更適合語音播放

4. 根據需求調整設定：
   - 如果主要使用文本交流，可以只啟用 TTS 功能
   - 如果主要使用語音輸入，可以只啟用 ASR 功能
   - 如果需要完整的語音對話體驗，啟用語音通話功能

希望這份使用說明能幫助您充分利用 Cherry Studio 的語音功能，享受更自然、便捷的 AI 互動體驗！
```
