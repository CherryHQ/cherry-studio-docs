# macOS

1.  首先到官網下載頁面點擊下載 Mac 版本，或點擊下方直達

請注意下載 **自己 Mac 對應的晶片版本**
{% hint style="info" %}
如果不知道自己的 Mac 應當使用什麼晶片版本：

*   點擊 Mac 左上角選單欄 
*   在展開選單中點擊「關於這台 Mac」
*   在彈出視窗中查看處理器資訊

如果為 Intel 晶片則下載 Intel 版本安裝包

如果為 Apple M\* 晶片則下載 Apple 晶片安裝包
{% endhint %}
