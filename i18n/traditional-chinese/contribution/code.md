# 貢獻程式碼

我們歡迎對 Cherry Studio 的貢獻！您可以透過以下方式貢獻：

1\. **貢獻程式碼：** 開發新功能或優化現有程式碼。

2\. **修復錯誤：** 提交您發現的錯誤修復。

3\. **維護問題：** 協助管理 GitHub 問題。

4\. **產品設計：** 參與設計討論。

5\. **撰寫文件：** 改進使用者手冊和指南。

6\. **社群參與：** 加入討論並幫助使用者。

7\. **推廣使用：** 宣傳 Cherry Studio。
