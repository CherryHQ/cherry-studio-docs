# macOS

1. Najpierw przejdź na oficjalną stronę pobierania, kliknij, aby pobrać wersję na Maca, lub kliknij poniższy bezpośredni link

Pobierz **wersję zgodną z chipem Twojego Maca**
{% hint style="info" %}
Jeśli nie wiesz, którą wersję chipa wybrać dla swojego Maca:

* Kliknij menu  w lewym górnym rogu Maca
* W rozwijanym menu kliknij "Informacje o tym Macu"
* W oknie dialogowym sprawdź informacje o procesorze

Jeśli masz chip Intel, pobierz pakiet instalacyjny wersji Intel

Jeśli masz chip Apple M\*, pobierz pakiet instalacyjny wersji Apple Silicon
{% endhint %}
