# आवाज़ सम्बंधी कार्यक्षमता

```
Cherry Studio वॉइस सुविधा उपयोग निर्देश

1. वॉइस कार्यक्षमता अवलोकन

Cherry Studio तीन प्रमुख वॉइस फंक्शन मॉड्यूल प्रदान करता है: TTS (टेक्स्ट टू स्पीच), ASR (ऑटोमैटिक स्पीच रिकग्निशन) और वॉइस कॉल। ये फीचर्स आपको ऑडियो के माध्यम से एआई के साथ प्राकृतिक संवाद करने में सक्षम बनाते हैं, जिससे उपयोगकर्ता अनुभव बेहतर होता है।

- TTS (टेक्स्ट टू स्पीच): एआई के जवाब को टेक्स्ट से ऑडियो में बदलता है
- ASR (ऑटोमैटिक स्पीच रिकग्निशन): आपकी आवाज़ को टेक्स्ट में बदलता है
- वॉइस कॉल: TTS और ASR को जोड़कर, चैटजीपीटी जैसा वॉइस कॉन्वर्सेशन अनुभव

2. TTS (टेक्स्ट टू स्पीच) फंक्शन

1. समर्थित सेवा प्रकार

Cherry Studio चार प्रकार की TTS सेवाएँ समर्थित करता है:

- OpenAI: OpenAI के TTS API का उपयोग करता है, API कुंजी आवश्यक
- ब्राउज़र TTS: ब्राउज़र की अंतर्निहित स्पीच सिंथेसिस सुविधा, मुफ़्त, कोई कॉन्फ़िगरेशन नहीं
- सिलिकॉनफ्लो: सिलिकॉनफ्लो (Siliconflow) की TTS सेवा, API कुंजी आवश्यक
- फ्री ऑनलाइन TTS: मुफ़्त ऑनलाइन TTS सेवा, API कुंजी की आवश्यकता नहीं

2. सेटअप विधि

1) सेटिंग्स पेज पर जाएँ, "वॉइस फंक्शन" टैब चुनें
2) "TTS" उपटैब में:
   - TTS सुविधा सक्षम करें (स्विच ऑन करें)
   - TTS सेवा प्रकार चुनें
   - चुनी गई सेवा के अनुसार संबंधित पैरामीटर्स कॉन्फ़िगर करें:
     - OpenAI: API कुंजी भरें, API एड्रेस, आवाज़ और मॉडल चुनें
     - ब्राउज़र TTS: आवाज़ चुनें
     - सिलिकॉनफ्लो: API कुंजी भरें, API एड्रेस, आवाज़, मॉडल, रिस्पॉन्स फॉर्मेट और स्पीड चुनें
     - फ्री ऑनलाइन TTS: आवाज़ और आउटपुट फॉर्मेट चुनें
3) TTS फ़िल्टर विकल्प कॉन्फ़िगर करें (वैकल्पिक):
   - सोच प्रक्रिया फ़िल्टर करें
   - मार्कडाउन टैग फ़िल्टर करें
   - कोड ब्लॉक फ़िल्टर करें
4) TTS प्रगति पट्टी दिखाना सेट करें
5) "TTS टेस्ट करें" बटन पर क्लिक करके कॉन्फ़िगरेशन जाँचें

3. उपयोग विधि

- TTS सक्षम होने पर, एआई का उत्तर स्वतः ऑडियो में बदल जाएगा
- चैट इंटरफ़ेस में प्रत्येक एआई जवाब के नीचे TTS प्ले बटन दिखेगा
- प्ले बटन पर क्लिक करके ऑडियो चला/रोक सकते हैं
- TTS प्रगति पट्टी सक्षम होने पर पाठ के नीचे प्रगति दिखेगी
- लंबा पाठ स्वतः खंडों में विभाजित होकर सतत चलेगा

3. ASR (ऑटोमैटिक स्पीच रिकग्निशन) फंक्शन

1. समर्थित सेवा प्रकार

Cherry Studio तीन प्रकार की ASR सेवाएँ समर्थित करता है:

- OpenAI: OpenAI के Whisper मॉडल का उपयोग, API कुंजी आवश्यक
- ब्राउज़र: ब्राउज़र की अंतर्निहित वॉइस रिकग्निशन सुविधा, मुफ़्त, कोई कॉन्फ़िगरेशन नहीं
- लोकल सर्वर: स्थानीय वेबसॉकेट सर्वर से कनेक्ट करके वॉइस रिकग्निशन

2. सेटअप विधि

1) सेटिंग्स पेज पर जाएँ, "वॉइस फंक्शन" टैब चुनें
2) "ASR" उपटैब में:
   - ASR सुविधा सक्षम करें (स्विच ऑन करें)
   - ASR सेवा प्रकार चुनें
   - चुनी गई सेवा के अनुसार संबंधित पैरामीटर्स कॉन्फ़िगर करें:
     - OpenAI: API कुंजी भरें, API एड्रेस, मॉडल चुनें
     - ब्राउज़र: कोई अतिरिक्त कॉन्फ़िगरेशन नहीं
     - लोकल सर्वर: एप्लिकेशन स्टार्ट होने पर स्वतः ASR सर्वर शुरू करना सेट कर सकते हैं
   - वॉइस रिकग्निशन भाषा चुनें (डिफ़ॉल्ट चीनी)
3) "ASR टेस्ट करें" बटन पर क्लिक करके कॉन्फ़िगरेशन जाँचें

3. उपयोग विधि

- ASR सक्षम होने पर, इनपुट बॉक्स के बगल में वॉइस रिकग्निशन बटन दिखेगा
- वॉइस रिकग्निशन बटन पर क्लिक करके रिकॉर्डिंग शुरू करें
- बोलने के बाद, आवाज़ टेक्स्ट में बदलकर इनपुट बॉक्स में भर जाएगी
- बटन पर फिर क्लिक करके रिकॉर्डिंग समाप्त करें
- वॉइस रिकग्निशन कई वाक्यों की सतत पहचान करता है, क्यूम्युलेटिव मोड में काम करता है

4. वॉइस कॉल फंक्शन

1. विशेषताएँ

- TTS और ASR को जोड़कर, चैटजीपीटी जैसा वॉइस डायलॉग अनुभव
- ड्रैग करने योग्य फ़्लोटिंग विंडो इंटरफ़ेस
- होल्ड टू टॉक मोड सपोर्ट
- कस्टम शॉर्टकट सपोर्ट
- विंडो कॉलैप्स सपोर्ट
- समर्पित वॉइस कॉल मॉडल चुन सकते हैं
- कस्टम प्रॉम्प्ट सपोर्ट

2. सेटअप विधि

1) सेटिंग्स पेज पर जाएँ, "वॉइस फंक्शन" टैब चुनें
2) "कॉल फंक्शन" उपटैब में:
   - वॉइस कॉल सुविधा सक्षम करें (स्विच ऑन करें)
   - "मॉडल चुनें" बटन पर क्लिक करके वॉइस कॉल के लिए एआई मॉडल चुनें
   - प्रॉम्प्ट बॉक्स में कस्टम वॉइस कॉल प्रॉम्प्ट डालें (वैकल्पिक)
   - "सेव करें" बटन पर क्लिक कर प्रॉम्प्ट सेव करें, या "रीसेट" बटन से डिफ़ॉल्ट प्रॉम्प्ट रिस्टोर करें

3. उपयोग विधि

1) चैट इंटरफ़ेस में, इनपुट बॉक्स के दाएँ वॉइस कॉल बटन (फोन आइकन) पर क्लिक करें
2) वॉइस कॉल विंडो खुलेगी, वेलकम ऑडियो चलेगा
3) "होल्ड टू टॉक" बटन को दबाए रखकर रिकॉर्डिंग शुरू करें (या सेट शॉर्टकट का उपयोग करें)
4) बटन छोड़कर रिकॉर्डिंग ख़त्म करें और एआई को भेजें
5) एआई जवाब जेनरेट करके TTS द्वारा चलाएगा
6) विंडो में नियंत्रण बटन:
   - म्यूट/अनम्यूट बटन: TTS आउटपुट नियंत्रित करें
   - पॉज़/कंटिन्यू बटन: संवाद रोकें या जारी रखें
   - सेटिंग बटन: शॉर्टकट कॉन्फ़िगर करें
   - कॉलैप्स बटन: विंडो समेटें, केवल "होल्ड टू टॉक" पंक्ति दिखेगी
7) क्लोज बटन पर क्लिक करके कॉल समाप्त करें

4. शॉर्टकट सेटअप

1) वॉइस कॉल विंडो में, सेटिंग बटन पर क्लिक करें
2) सेटिंग पैनल में, शॉर्टकट बटन पर क्लिक करें
3) वांछित की दबाएँ (जैसे स्पेसबार, शिफ्ट आदि)
4) "सेव करें" बटन पर क्लिक करके सेटिंग सेव करें
5) उपयोग में, सेट शॉर्टकट दबाकर रिकॉर्डिंग शुरू करें, छोड़कर भेजें

5. सामान्य समस्याएँ और समाधान

1. TTS संबंधी समस्याएँ

- समस्या: TTS से आवाज़ नहीं आती
  समाधान: जाँचें कि TTS सक्षम है, सही सेवा प्रकार चुना है और आवश्यक पैरामीटर्स भरे हैं

- समस्या: TTS आवाज़ गुणवत्ता ख़राब है
  समाधान: भिन्न TTS सेवा प्रकार या आवाज़ ट्राई करें

- समस्या: TTS चलते समय त्रुटि संदेश
  समाधान: API कुंजी सही है क्या जाँचें, नेटवर्क कनेक्शन सही है क्या देखें

2. ASR संबंधी समस्याएँ

- समस्या: ASR आवाज़ नहीं पहचानता
  समाधान: जाँचें ASR सक्षम है, सही सेवा प्रकार चुना है और आवश्यक पैरामीटर्स भरे हैं

- समस्या: ASR पहचान सटीकता कम
  समाधान: भिन्न ASR सेवा प्रकार ट्राई करें, या माइक्रोफ़ोन पोज़िशन और वॉल्यूम एडजस्ट करें

- समस्या: ASR सर्वर कनेक्शन विफल
  समाधान: लोकल सर्वर ठीक चल रहा है क्या देखें, या एप्लिकेशन रीस्टार्ट करें

3. वॉइस कॉल संबंधी समस्याएँ

- समस्या: वॉइस कॉल विंडो नहीं खुलती
  समाधान: जाँचें वॉइस कॉल सक्षम है, TTS और ASR सही कॉन्फ़िगर हैं

- समस्या: होल्ड टू टॉक रिस्पॉन्स नहीं देता
  समाधान: माइक्रोफ़ोन परमिशन चेक करें, या वॉइस कॉल रीस्टार्ट करें

- समस्या: एआई जवाब में आवाज़ नहीं आती
  समाधान: जाँचें TTS सक्षम है, म्यूट नहीं है

6. उन्नत सेटिंग्स और कस्टम विकल्प

1. TTS उन्नत सेटिंग्स

- फ़िल्टर विकल्प: सोच प्रक्रिया, मार्कडाउन टैग और कोड ब्लॉक फ़िल्टर कर सकते हैं, TTS प्लेबैक स्मूथ होगा
- प्रगति पट्टी प्रदर्शन: TTS प्रगति पट्टी दिखाना चुना जा सकता है
- कस्टम आवाज़ और मॉडल: कस्टम आवाज़ और मॉडल विकल्प जोड़ सकते हैं

2. ASR उन्नत सेटिंग्स

- स्वतः सर्वर स्टार्ट: एप्लिकेशन स्टार्ट होने पर ASR सर्वर स्वतः स्टार्ट करना सेट कर सकते हैं
- भाषा चयन: विभिन्न वॉइस रिकग्निशन भाषाएँ चुन सकते हैं

3. वॉइस कॉल उन्नत सेटिंग्स

- कस्टम प्रॉम्प्ट: वॉइस कॉल मोड में एआई को निर्देश देने वाले कस्टम प्रॉम्प्ट डाल सकते हैं
- समर्पित मॉडल चयन: वॉइस कॉल के लिए अलग एआई मॉडल चुन सकते हैं, वर्तमान संवाद से अलग
- शॉर्टकट कस्टमाइज़ेशन: रिकॉर्डिंग नियंत्रित करने वाले कस्टम शॉर्टकट सेट कर सकते हैं

7. उपयोग सुझाव

1. उचित TTS सेवा चुनें:
   - उच्च गुणवत्ता आवाज़ के लिए OpenAI या सिलिकॉनफ्लो सुझाएँ जाते हैं
   - API कॉन्फ़िगर न करना हो तो ब्राउज़र TTS या फ्री ऑनलाइन TTS उपयोग करें

2. उचित ASR सेवा चुनें:
   - उच्च सटीकता के
