# कोड संरचना

```yaml
...
├─ docs/ # दस्तावेज़ निर्देशिका
├─ resources/ # संसाधन फ़ाइल निर्देशिका
├─ scripts/ # स्क्रिप्ट फ़ाइल निर्देशिका
└─ src/ # मुख्य स्रोत कोड निर्देशिका
    ├─main/ # मुख्य प्रक्रिया कोड
    ├─preload/ # प्रीलोड स्क्रिप्ट निर्देशिका
    └─renderer/ # रेंडरिंग प्रक्रिया कोड
        ├─src/ # रेंडरिंग प्रक्रिया का स्रोत कोड
            ├─assets/ # संसाधन फ़ाइलें
                ├─fonts/ # फ़ॉन्ट फ़ाइलें
                ├─images/ # प्रोफ़ाइल चित्र और अन्य छवि फ़ाइलें
                └─styles/ # शैली फ़ाइलें
            ├─components/ # घटक
            ├─config/ # कॉन्फ़िग फ़ाइलें
            ├─context/ # संदर्भ
            ├─databases/ # डेटाबेस संबंधित फ़ाइलें
            ├─hooks/ # कस्टम हुक्स
            ├─i18n/ # अंतरराष्ट्रीयकरण फ़ाइलें
            ├─pages/ # पेज फ़ाइलें
            ├─providers/ # सेवा प्रदाता संबंधी कॉन्फ़िग
            ├─services/ # सेवा फ़ाइलें
            ├─store/ # स्थिति प्रबंधन फ़ाइलें
            ├─types/ # TypeScript प्रकार परिभाषा फ़ाइलें
            ├─utils/ # उपयोगिता कार्य
            ...
        ...
    ...
...
```
